# <a name="irm-protection-to-sharepoint-files"></a>A tartalomvédelmi szolgáltatás védelme a SharePoint-fájlok


<p>SharePoint Online belül tartalomvédelemmel fájlokat a lista- és dokumentumtár szintjén alkalmazzák. Mielőtt a szervezet tartalomvédelemmel használni, először be kell állítania a tartalomvédelmi szolgáltatás. A tartalomvédelmi szolgáltatás a Azure tartalomvédelmi szolgáltatás Azure információvédelem titkosítására, és rendelje hozzá a használati korlátozások támaszkodik. Néhány Office 365 terveknek tartalmazniuk, Azure Rights Management, de nem mindegyik. További, olvassa el a <a href="https://docs.microsoft.com/azure/information-protection/understand-explore/office-apps-services-support" data-linktype="external">hogyan Office alkalmazások és szolgáltatások Azure tartalomvédelmi szolgáltatás támogatja</a>.</p> <ul> <li><a href="https://docs.microsoft.com/en-us/office365/securitycompliance/set-up-irm-in-sp-admin-center">Set up Information Rights Management (IRM) in SharePoint admin center</a></li> <li><a href="https://docs.microsoft.com/en-us/office365/securitycompliance/set-up-irm-in-sp-admin-center#irm-enable-sharepoint-document-libraries-and-lists">A tartalomvédelmi szolgáltatás engedélyezése SharePoint-dokumentumtárak és listák</a></li> <li><a href="https://support.office.com/en-US/Article/Information-Rights-Management-in-Office-c7a70797-6b1e-493f-acf7-92a39b85e30c">Az Office tartalomvédelmi szolgáltatása</a></li> <li><a href="https://docs.microsoft.com/en-us/office365/SecurityCompliance/information-rights-management-in-exchange-online">Az Exchange Online tartalomvédelmi szolgáltatás</a></li> </ul>
