# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Megjelenítési nyelvi beállítások módosítása a SharePoint-vagy OneDrive

Nyelvi beállítások hibaelhárításával kapcsolatos további tudnivalókért lásd: <a title="nem tudja módosítani az alapértelmezett nyelv egy SharePoint Online webhelycsoporthoz" href="https://support.office.com/en-us/article/you-can-t-change-the-default-language-for-a-sharepoint-online-site-collection-40dda07e-6b41-49e9-9828-8805dcb92964">Nem módosíthatja az alapértelmezett nyelv a SharePoint Online webhelycsoport</a>.</li> </ol> <p>&nbsp;</p>
