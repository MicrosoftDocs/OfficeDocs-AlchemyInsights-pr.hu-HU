# <a name="failed-to-create-the-plan-error-when-trying-to-create-a-planner-tab-in-microsoft-teams"></a>"Nem sikerült létrehozni a tervet" hibaüzenet jelenik meg, amikor a Microsoft teamben egy tervező lapot próbál létrehozni

Ha "a terv létrehozása sikertelen" hibaüzenetet kapja, amikor a Microsoft teamben egy tervező lapot próbál létrehozni, győződjön meg arról, hogy az adminisztrátor tagja a csoportnak az alábbi lépések végrehajtásával:

1. Az admin központban lépjen a **csoportok** > [csoportokra.](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/groups) 
2. Jelölje ki a csoportot, majd a **tagok** lapon jelölje be az **összes megtekintése és a tagok kezelése**választógombot.
3. Adja hozzá a tagot, és válassza a **Mentés**.
