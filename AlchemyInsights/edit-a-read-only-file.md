# <a name="edit-a-read-only-file"></a>Írásvédett fájl szerkesztése

Ha a dokumentumtárakban a dokumentum nevére kattintunk, előfordulhat, hogy az Office-program váratlanul csak olvasható fájlként nyitja meg. Szerkesztésre való megnyitásához kattintson a **Megnyitás** gombra (a három pont), majd kattintson a **Szerkesztés gombra.**
