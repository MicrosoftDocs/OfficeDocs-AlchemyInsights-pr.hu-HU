# <a name="edit-a-read-only-file"></a>Csak olvasható fájl szerkesztése

A dokumentumtárak kattintva a dokumentum nevére, az Office-program lehet, hogy váratlanul nyissa meg a csak olvasható fájlként. Nyissa meg szerkesztésre, **Nyissa meg a** (három pont), és kattintson a **Szerkesztés**
