# <a name="application-protection-policy-requirements"></a>Alkalmazásvédelmi házirend követelményei

Alkalmazásvédelmi házirendkövetelmények:

- A felhasználónak Intune- vagy EMS-licence van.
- A felhasználó egy alkalmazásvédelmi házirendekkel megcélzott csoportba tartozik.
- Egy eszközön csak egy vállalati felhasználó van bejelentkezve a védett alkalmazásokba.
- [Az alkalmazás végrehajtotta az Intune SDK-t. Az SDK-t támogató appok listáját a Microsoft Intune által védett alkalmazásokban láthatja.](https://docs.microsoft.com/mem/intune/apps/apps-supported-intune-apps)
