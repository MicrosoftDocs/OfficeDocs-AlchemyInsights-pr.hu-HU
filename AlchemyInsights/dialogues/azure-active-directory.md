# <a name="azure-active-directory"></a>Azure Active Directory

A Microsoft Azure Active Directory (Azure AD) egy felhőalapú identitás- és hozzáférés-kezelési szolgáltatás, amely segítséget nyújt az alkalmazottaknak a bejelentkezésben és az erőforrásokhoz való hozzáférésben külsőleg és belsőleg egyaránt. További információért olvassa el az alábbi forrásokat:

- [Mi az Azure Active Directory?](https://go.microsoft.com/fwlink/?linkid=2081145)
- [Azure-integráció az Office 365-ben](https://go.microsoft.com/fwlink/?linkid=2081218)
- [Azure Active Directory ingyenes próbaverzió](https://go.microsoft.com/fwlink/?linkid=2081144)
