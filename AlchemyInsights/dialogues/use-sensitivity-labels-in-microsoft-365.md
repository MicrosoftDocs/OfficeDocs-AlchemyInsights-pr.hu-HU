# <a name="use-sensitivity-labels-in-microsoft-365"></a>Bizalmasság-címkék használata a Microsoft 365-ben

A bizalmasság-címkékkel:
- Védelmi beállításokat, például titkosítást vagy vízjeleket kényszerítsen a címkével jelölt tartalmakon.
- Az Office-appok tartalmának védelme különböző platformokon és eszközökön.
- Külső alkalmazásokban és szolgáltatásokban található tartalmak védelme a Microsoft Cloud App Security használatával.
- A bizalmasság-címkék kiterjesztése külső alkalmazásokra és szolgáltatásokra.
- A tartalmakat a védelmi beállítások használata nélkül osztályozhatja.

További információt a [bizalmasság-címkékről olvashat.](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels)
