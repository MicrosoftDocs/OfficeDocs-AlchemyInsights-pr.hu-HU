# <a name="permissions-and-browser-related-requirements-for-microsoft-365-defender"></a>A Microsoft 365 Defender engedélyekkel és böngészővel kapcsolatos követelményei

- Engedélyek: A Microsoft 365 Defender bekapcsolásához globális rendszergazdának vagy az Azure Active Directory biztonsági rendszergazdának kell lennie. A Microsoft 365 Defendert használhatja szerepkörök listájáról, valamint az adatokhoz való hozzáférés szabályozásáról a [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2143626)elérésének kezelése szolgáltatásban olvashat.

- A böngészővel kapcsolatos követelmények: A Microsoft 365 Defender Microsoft 365 biztonsági központban való eléréséhez használja a Microsoft Edge-et, az Internet Explorer 11-et vagy bármilyen 5-ös HTML-kompatibilis webböngészőt.
