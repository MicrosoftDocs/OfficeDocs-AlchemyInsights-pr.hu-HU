# <a name="public-folder-access-from-mobile-devices"></a>Nyilvános mappák elérése mobileszközökön

A nyilvános mappákhoz való hozzáférés nem használható mobileszközökön, kérjük, hogy az OWA (webes Outlook) vagy az asztali Windows Outlook vagy a Mac Outlook ügyfelei hozzáférhessenek a nyilvános mappákhoz.

Ha a nyilvános mappákhoz hasonló funkciókat szeretne használni, és a tartalmakat mobil eszközökön is elérhetővé szeretné tenni, a jobb alternatíva érdekében tanulmányozza a [Microsoft 365-csoportok ismertetése](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) című témakört.
