# <a name="create-a-new-folder"></a>Új mappa létrehozása

1. A **mappák** a navigációs ablakban jelölje ki az **új mappát**. 
    
2. Írja be a mappa nevét, és nyomja le az ENTER billentyűt.
    
További információért lásd [az Outlook.com mappa létrehozása](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).
