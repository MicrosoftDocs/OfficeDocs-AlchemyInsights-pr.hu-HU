# <a name="using-the-microsoft-authenticator-app"></a>A Microsoft hitelesítő app használata

Bejelentkezhet személyes Microsoft-fiókjába ujjlenyomat, arcfelismerő vagy PIN-kód használatával.

- [A Microsoft hitelesítő app használata](https://support.microsoft.com/help/4026727/microsoft-account-how-to-use-the-microsoft-authenticator-app) 

- [Beállíthatja a hitelesítő alkalmazást](https://docs.microsoft.com/azure/active-directory/user-help/security-info-setup-auth-app) a QR-kód beolvasásával vagy [egy fiók manuális hozzáadásával](https://docs.microsoft.com/azure/active-directory/user-help/user-help-auth-app-add-account-manual).  

További információt a [hitelesítő app áttekintése](https://docs.microsoft.com/azure/active-directory/user-help/user-help-auth-app-overview)című témakörben talál.
