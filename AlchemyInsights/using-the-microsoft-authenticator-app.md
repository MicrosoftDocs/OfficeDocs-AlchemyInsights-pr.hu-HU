# <a name="using-the-microsoft-authenticator-app"></a>A Microsoft Authenticator app használata

Jelentkezzen be személyes Microsoft-fiókjába ujjlenyomat, arcfelismerés vagy PIN-kód használatával.

- [A Microsoft Authenticator app használata.](https://support.microsoft.com/help/4026727/microsoft-account-how-to-use-the-microsoft-authenticator-app) 

- [Az Authenticator app beállításához](https://docs.microsoft.com/azure/active-directory/user-help/security-info-setup-auth-app) vagy beolvassa a QR-kódot, vagy [manuálisan fel kell vennie egy fiókot.](https://docs.microsoft.com/azure/active-directory/user-help/user-help-auth-app-add-account-manual)  

További információt Az [Authenticator app](https://docs.microsoft.com/azure/active-directory/user-help/user-help-auth-app-overview)áttekintése lapon talál.
