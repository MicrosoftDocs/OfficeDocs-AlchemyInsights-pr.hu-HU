# <a name="restore-files-or-folders-from-the-sharepoint-recycle-bin"></a>Fájlok vagy mappák visszaállítása a SharePoint Lomtárból 

Amikor töröl egy elemet egy SharePoint-dokumentumtárból vagy -listából, az nem törlődik azonnal a SharePointból. A törölt elemek egy ideig a webhely lomtárába kerülnek. Ez alatt az idő alatt visszaállíthatja őket az eredeti helyükre.
