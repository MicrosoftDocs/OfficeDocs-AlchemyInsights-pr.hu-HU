# <a name="screen-sharing-with-skype-consumer-users"></a>Képernyőmegosztás a Skype fogyasztói felhasználóival

A program megosztásának általános útmutatását, a megosztás korlátozásait és a hibaelhárítási forrásokat A képernyő megosztása a Skype Vállalati verzióban című [témakörben olvashatja](https://support.microsoft.com/office/share-and-present-content-from-skype-meetings-app-skype-for-business-web-app-234b0c06-a88d-4707-904c-4fd6c571fc01)el.  

A Skype Vállalati verzió felhasználói jelenleg nem tudnak képernyő-megosztást megosztani a skype fogyasztói felhasználóival. Ide [kattintva visszajelzést](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) küldhet a képernyőmegosztás engedélyezéséről a fogyasztói Skype-felhasználókkal.
