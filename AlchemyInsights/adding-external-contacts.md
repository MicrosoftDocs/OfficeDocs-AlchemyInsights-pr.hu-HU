# <a name="enable-external-communications-and-add-contacts"></a>Külső kommunikáció engedélyezése és névjegyek felvétele

A Skype Vállalati verzióban személyeket is felvehet a partnerlistájára, így egy kattintással elérhetők lesznek. A program minden felvett partnert besorol egy vagy több partnercsoportba. További információért olvassa el [a Partner felvétele a Skype Vállalati verzióban](https://support.office.com/article/add-a-contact-in-skype-for-business-89338023-2adf-4f5c-90b6-f8b6f72fadd1) című részt. 

Győződjön meg arról, hogy a bérlői rendszergazda követte a [külső kommunikáció engedélyezése](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users)lépéseket.

Csak azt szeretné, hogy a felhasználók kapcsolatba lépjenek a Skype fogyasztói felhasználóival? Lásd: [A Skype Vállalati verzió felhasználói skype-névjegyek felvétele](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts).
