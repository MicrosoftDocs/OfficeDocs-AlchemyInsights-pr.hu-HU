# <a name="cant-add-default-2010-approval-workflow"></a>Nem lehet hozzáadni az alapértelmezett jóváhagyási munkafolyamat 2010

A Microsoft SharePoint-webhelycsoport nem adhat (például a "Jóváhagyás – SharePoint 2010") globálisan újrafelhasználható munkafolyamat listához vagy könyvtárhoz. A jóváhagyási munkafolyamat hiányozhat. A probléma megoldásához hivatkozik az [nem adható hozzá a SharePoint-lista vagy tár globálisan újrafelhasználható munkafolyamat](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

Munkafolyamat-jóváhagyások kapcsolatos további információért lásd: [mindent jóváhagyási munkafolyamatok] (https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742). 
 
Tudjon meg többet a [Modern jóváhagyási folyamat] (https://flow.microsoft.com/blog/introducing-modern-approvals).
