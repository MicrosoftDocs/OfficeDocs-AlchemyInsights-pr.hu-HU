# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Outlook-hiba: Nem indítható a Microsoft Outlook

Nem sikerül elindítani az Outlook 2016-ot, az Outlook 2013-at vagy az Outlook 2010-et, vagy nem tudja elindítani a "Nem indítható a Microsoft Outlook. Nem lehet megnyitni az Outlook ablak" hibaüzeneteket?
  
Először használja a [cikk](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) hibaelhárítási lépéseit az indítási problémák megoldásához. 
  
A [Speciális diagnosztika eszközt](https://aka.ms/SaRA-OutlookAdvDiagnostics) az érintett gépen is futtathatja.
