# <a name="map-a-sharepoint-library-to-a-network-drive"></a>SharePoint-tár leképezése hálózati meghajtóra

A tárak hálózati meghajtóként való hozzárendelése csak az Internet Exploreren keresztül lehetséges. Ehelyett [szinkronizálja a SharePoint-fájlokat az új OneDrive szinkronizáló ügyféllel](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx), amely [igény szerinti fájlokat](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx)biztosít. Belépés minden-a fájlokat-ban OneDrive nélkül használ helyi raktározás hely.
  
[Segítséget kaphat a csatlakoztatott hálózati meghajtók konfigurálásában és hibaelhárításánál](https://go.microsoft.com/fwlink/?linkid=872946)
