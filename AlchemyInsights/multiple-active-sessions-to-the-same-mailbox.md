# <a name="multiple-active-sessions-to-the-same-mailbox"></a>Több aktív munkamenet ugyanabba a postaládába

Az Exchange-erőforrások használatának szabályozásához a postaládának "költségvetése" van.

A költségvetésen felüli kivételt a következő körülmények válthatják ki, de nem kizárólagosan:

- Ugyanabban az Outlook Web App-munkamenetben néhány böngészőlap nyílik meg.

- Néhány aktív Outlook Web App-munkamenet ugyanarra a postaládára.

- Néhány más ügyfélalkalmazás (Outlook, Outlook Mobile, egy harmadik féltől származó ügyfélalkalmazás) egyszerre fér hozzá a postaládához.

- A hosszú ideig futó műveletek, például a keresési kérelmek végrehajtása egy másik aktív postaláda-munkamenetből történik.
