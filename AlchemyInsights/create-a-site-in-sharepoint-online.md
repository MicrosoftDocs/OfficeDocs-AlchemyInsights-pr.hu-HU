# <a name="create-a-site"></a>Webhely létrehozása

A SharePoint Online-ban különböző típusú webhelyeket hozhat létre. Ha tudni szeretné, hogyan teheti ezt meg az új SharePoint Felügyeleti központban, olvassa el a Webhely létrehozása című [témakört.](https://go.microsoft.com/fwlink/?linkid=866295) Ha tudni szeretné, hogy miként hozhatják létre a felhasználók saját webhelyeiket, olvassa el [a Webhelylétrehozásának kezelése a SharePoint Online-ban című témakört.](https://go.microsoft.com/fwlink/?linkid=866296)
 
Az új webhelytípusokról a [Csatlakoztatott csoporthelyek](https://go.microsoft.com/fwlink/?linkid=866292) csoportosítása vagy [a kommunikációs helyek című témakörben talál](https://go.microsoft.com/fwlink/?linkid=866294)további információt.
