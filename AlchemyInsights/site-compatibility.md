# <a name="learn-about-site-compatibilityaffecting-changes-coming-to-microsoft-edge"></a>A webhelyek kompatibilitásának ismertetése – a Microsoft Edge-re érkező módosítások hatása

Az Internet folyamatosan fejlődik a felhasználói élmény, a biztonság és az adatvédelem javítása érdekében. Egyes módosítások elég jelentősek a meglévő lapok működésének befolyásolásához. A Microsoft Edge csapat frissíti az oldalakat a gondolkodás előrehaladtával, a megszilárdult idősorokkal és a [változtatások bejelentésével](https://go.microsoft.com/fwlink/?linkid=2135534).
