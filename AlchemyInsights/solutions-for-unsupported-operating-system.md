# <a name="solutions-for-unsupported-operating-system"></a>A "nem támogatott operációs rendszer" megoldások

Amikor megpróbálja telepíteni a Microsoft 365-et, egy nem **támogatott operációs rendszerről** tájékoztató hibaüzenet jelenhet meg, ha az eszköz nem felel meg a [rendszerkövetelményeknek](https://products.office.com/office-system-requirements).
  
A Microsoft 365 a következőket igényli:
  
- PC: Windows 10, Windows 8,1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 vagy Windows Server 2008 R2

    VAGY

- Mac OS X 10,10

Ha a következő üzenet jelenik meg, miközben az Office-t mobileszközön vagy Chromebookon kísérli meg telepíteni, olvassa el a [Microsoft Office telepítése és futtatása Chromebookon-on](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA)című témakört.
