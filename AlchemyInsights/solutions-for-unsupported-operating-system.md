# <a name="solutions-for-unsupported-operating-system"></a>Megoldások a "Nem támogatott operációs rendszerhez"

A Microsoft 365 telepítésekor előfordulhat, hogy **az operációs rendszer nem támogatott** hibaüzenetjelenik meg, ha az eszköz nem felel meg a [rendszerkövetelményeknek.](https://products.office.com/office-system-requirements)
  
A Microsoft 365 használatához a következőkre van szükség:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 vagy Windows Server 2008 R2

    VAGY

- Mac OS X 10.10

Ha ez az üzenet akkor jelenik meg, amikor mobileszközre vagy Chromebookra próbálja telepíteni az Office-t, olvassa [el a Microsoft Office telepítése és futtatása Chromebookon című témakört.](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA)
