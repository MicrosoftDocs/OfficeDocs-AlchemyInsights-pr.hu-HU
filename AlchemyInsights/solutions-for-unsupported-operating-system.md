"Nem támogatott operációs rendszer"

# <a name="solutions-for-unsupported-operating-system"></a>"Nem támogatott operációs rendszer" megoldások

Amikor megpróbálja telepíteni az Office 365/2016, a következő hibaüzenet jelenhet meg, ha az eszköz nem felel meg a [Rendszerkövetelmények](https://products.office.com/office-system-requirements).
  
 **"Nem támogatott operációs rendszer"**
  
A Microsoft Office 365/2016 használatához:
  
PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 vagy Windows Server 2008 R2
  
VAGY
  
Mac OS X 10.10
  
Ha ezt az üzenetet kapott egy mobil eszköz vagy a Chromebook Office telepítése közben, lásd az [Office-alkalmazások és a mobil eszközön e-mail beállítása](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).
