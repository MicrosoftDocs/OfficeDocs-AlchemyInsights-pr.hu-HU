# <a name="solutions-for-unsupported-operating-system"></a>"Nem támogatott operációs rendszer" megoldások

Amikor megpróbálja telepíteni az Office 365, egy **nem támogatott operációs rendszer** hibaüzenet megjelenhet, ha az eszköz nem felel meg a [Rendszerkövetelmények](https://products.office.com/office-system-requirements).
  
A Microsoft Office 365 használatához:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 vagy Windows Server 2008 R2

    VAGY

- Mac OS X 10.10

Ha ezt az üzenetet kapott egy mobil eszköz vagy a Chromebook Office telepítése közben, lásd: [hogyan kell telepíteni és futtatni a Chromebook a Microsoft Office](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
