# <a name="desktop-app-for-sharepoint"></a>SharePoint-alapú asztali alkalmazás

A szervezet a SharePoint Desktop alkalmazás személyes előnézeti programjában szerepel. A Microsoft jelenleg nem támogatja a SharePoint rendszerhez való asztali alkalmazást (személyes előnézet). A SharePoint-alapú asztali alkalmazással kapcsolatos segítségért használja a Microsoft Private Preview saját partnereit.
