# <a name="enable-seamless-sso"></a>Zökkenőmentes egyszeri bejelentkezés engedélyezése

Azúrkék Active Directory (Azure AD) zökkenőmentes egyszeri bejelentkezés (Seamless SSO) automatikusan aláírja a felhasználókat, amikor a vállalati hálózathoz csatlakoztatott vállalati asztalokon vannak.
  
[A problémamentes egyszeri bejelentkezés (SSO) engedélyezéséhez kövesse az ebben a cikkben leírt lépéseket](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
