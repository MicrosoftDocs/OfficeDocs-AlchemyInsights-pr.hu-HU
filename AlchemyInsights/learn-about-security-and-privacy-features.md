# <a name="learn-about-the-security-and-privacy-features-of-microsoft-search-in-bing"></a>Tudnivalók a Microsoft Search biztonsági és adatvédelmi funkcióiról a Bingben

A Microsoft Search biztonsági és adatvédelmi funkciói a Bingben segítenek a szervezeteknek a felhasználói adatok és a munkahelyi adatok védelmében. A kérelmek például HTTPS-kapcsolaton keresztül, a kapcsolatok pedig a végpontok között titkosítva vannak.

További információ: A Microsoft Keresés biztonsági és adatvédelmi beállításai [a Bingben.](https://docs.microsoft.com/microsoftsearch/security-for-search)
