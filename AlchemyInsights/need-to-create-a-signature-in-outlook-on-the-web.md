# <a name="need-to-create-an-email-signature"></a>Létre kell hoznia egy e-mail-aláírást?

Ha van Outlook-fiókja vagy webes Outlookja (munkahelyi vagy iskolai) fiókja, létrehozhatja az aláírást, és automatikusan felveheti az összes kimenő üzenetbe, vagy szelektíven hozzáadhatja bizonyos kimenő üzenetekhez. Figyelje meg, hogy az e-mail-aláírása különbözik az Outlookban és a webes Outlookban. E-mail-aláírás létrehozásához olvassa el az [aláírás létrehozása és hozzáadása az üzenetekhez az Outlookban](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) vagy az [e-mail-aláírás létrehozása és hozzáadása a webes Outlookban](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)című témakört.
