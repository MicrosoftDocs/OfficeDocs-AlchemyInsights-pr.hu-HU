# <a name="install-and-configure-onedrive"></a>A OneDrive telepítése és konfigurálása

A OneDrive szinkronizálási ügyfélalkalmazás előre telepítve van a Windows 10-ben. A Windows 7 vagy Mac OneDrive telepítéséhez, illetve a mobilalkalmazás [https://onedrive.com/download](https://onedrive.com/download)telefonra vagy táblagépre való beszerzéséhez válassza a webhelyet.
  
A OneDrive szinkronizálási ügyfélalkalmazás Windows rendszerben való beállításához lásd:
  
- [A számítógép beállítása a OneDrive Vállalati verzió fájljainak szinkronizálására](https://go.microsoft.com/fwlink/?linkid=533375)

- [SharePoint-fájlok szinkronizálása a OneDrive új szinkronizálási ügyfelével](https://go.microsoft.com/fwlink/?linkid=871666)
