# <a name="learn-about-the-progressive-web-apps-pwas-on-microsoft-edge-chromium"></a>További információ a progresszív webalkalmazásról a Microsoft Edge-ben (Chromium)

A progresszív webalkalmazások a webhelyeket fokozatosan továbbfejlesztve úgy működnek, mint az őket támogató natív appok. A PWA-k kombinálják a web és a natív alkalmazások legjobb minőségét.

A webplatform szempontjából a Project Web Appok a [Microsoft Edge-ben (Chromium)](https://docs.microsoft.com/microsoft-edge/progressive-web-apps-chromium/#pwas-on-microsoft-edge-chromium) szabványokon alapulnak. Lehetővé teszik a felhasználóknak, hogy közvetlenül a böngészőből telepítsenek alkalmazásokat, és nincs szükségük Microsoft Store-alapú telepítésre vagy regisztrációra. Az asztali PWA-k minden olyan platformon támogatottak, ahol elérhető a Microsoft Edge (Chromium).
