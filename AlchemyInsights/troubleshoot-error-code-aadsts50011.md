# <a name="troubleshoot-error-code-aadsts50011"></a>Az AADSTS50011 hibakód hibaelhárítása

Az AADSTS50011-es hiba elhárításához végezze el az alábbi ajánlott lépést.

**AADSTS50011:** InvalidReplyTo – A válaszcím hiányzik, helytelenül van beállítva, vagy nem egyezik az apphoz konfigurált válaszcímekkel.

A probléma megoldásaként vegye fel ezt a hiányzó válaszcímet az Azure Active Directory (AD) appba, vagy vegyen fel valakit, aki rendelkezik az alkalmazás AD-ban való kezeléséhez szükséges engedélyekkel. További információt az [AADSTS50011-es](https://docs.microsoft.com/troubleshoot/azure/active-directory/error-code-aadsts50011-reply-url-mismatch)hibaelhárító cikkben talál.
