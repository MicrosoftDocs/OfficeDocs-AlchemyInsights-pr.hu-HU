# <a name="control-sync-throughput"></a>Vezérlés szinkronizálási átviteli-kontitára

Ha az alkalmazásforgalom szinkronizálását kell szabályoznia, azt javasoljuk, hogy lehetőség szerint használja a hálózati szolgáltatásminőségi (QoS) vagy a Windows QoS-házirendeket. A felhasználók számára elérhetővé tehet hálózati beállításokról [a OneDrive-szinkronizálási alkalmazás feltöltési vagy letöltési sebességének módosítása](https://support.office.com/article/71cc69da-2371-4981-8cc8-b4558bdda56e) című témakörben olvashat.
