# <a name="enable-versioning-for-a-list-or-library"></a>Verziószámozás engedélyezése egy listához vagy tárhoz

1. Nyissa meg a listát vagy tárat.
    
2. Kattintson a **Beállítások**gombra, majd a **tár beállításai** vagy a **lista beállításai**parancsra.
    
3. Kattintson a **Verziószámozási beállítások**hivatkozásra.
    
4. Adja meg a kívánt beállításokat, majd kattintson **az OK gombra**.
