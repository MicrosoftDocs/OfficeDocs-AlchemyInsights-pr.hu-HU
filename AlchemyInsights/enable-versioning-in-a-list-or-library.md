# <a name="enable-versioning-for-a-list-or-library"></a>A lista vagy tár verziószámozásának engedélyezése

1. Ugrás a lista vagy tár.
    
2. Kattintson a **Beállítások**gombra, és kattintson a **lista**vagy a **dokumentumtár beállításait** .
    
3. Kattintson a **Verziószámozási beállítások**gombra.
    
4. Válassza ki a kívánt beállításokat, és kattintson az **OK gombra**.
