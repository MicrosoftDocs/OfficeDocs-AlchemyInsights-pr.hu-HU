# <a name="unable-to-print"></a>Nem lehet nyomtatni

Tisztában vagyunk azzal a problémával, amely miatt a felhasználók nem tudják kinyomtatni a Safari vagy az Edge böngészőt. Miközben dolgozunk a javításon, kérjük, használjon másik böngészőt. Elnézést kérünk a kényelmetlenségért.
