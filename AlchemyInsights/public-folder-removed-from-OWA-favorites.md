# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Nyilvános mappa eltávolítva a Outlook (OWA) kedvencek közül

Egy ismert probléma miatt az Outlook Web App minden kijelentkezéskor vagy képernyőfrissítéskor eltávolítja a nyilvános mappákat a kedvencek közül. Ezt a problémát hamarosan kijavítjuk. Addig is adja hozzá újra a mappát az OWA alkalmazásból való hozzáféréshez.
