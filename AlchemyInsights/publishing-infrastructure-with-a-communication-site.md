# <a name="publishing-infrastructure-with-a-communication-site"></a>Az infrastruktúra közzététele a kommunikációs webhellyel


A SharePoint Server közzétételi infrastruktúra szolgáltatás engedélyezése a kommunikációs webhelyeken **nem támogatott** . 
  
További információt a [közzétételi helyek áthelyezése a kommunikációs webhelyekre](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience)című témakörben talál.
