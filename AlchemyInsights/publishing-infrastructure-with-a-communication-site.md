# <a name="publishing-infrastructure-with-a-communication-site"></a>Közzétételi webhely kommunikációs infrastruktúra

További információért lásd a [Moving a közzétételi webhelyeket hozzá kommunikációs helyek](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
