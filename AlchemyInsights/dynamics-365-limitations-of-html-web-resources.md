# <a name="webpage-html-web-resources-limitations"></a>A weblapok (HTML) webes erőforrásainak korlátai

* Egy HTML-alapú webes erőforrás nem tartalmazhat olyan kódot, amelyet a kiszolgálón végre kell hajtani. A ASP.NET lapokat nem lehet HTML webes erőforrásként feltölteni.

* A HTML-alapú webes erőforrások csak korlátozott számú lekérdezési karakterláncot tudnak elfogadni. További információ: [paraméterek a HTML webes erőforrásokhoz való átadással](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)

[Weboldal (HTML) webes erőforrások](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources)
