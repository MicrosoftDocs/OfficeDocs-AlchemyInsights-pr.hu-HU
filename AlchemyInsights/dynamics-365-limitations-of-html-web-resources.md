# <a name="webpage-html-web-resources-limitations"></a>Weboldal (HTML) webes erőforrások korlátai

* Egy HTML webes erőforrás nem tartalmazhat olyan kódot, amely a kiszolgálón végre kell hajtani. Az ASP.NET-lapok HTML webes erőforrásként nem tölthető fel.

* HTML webes erőforrások csak korlátozott számú lekérdezési karakterlánc paramétereinek képesek fogadni. További információ: [Paraméterek HTML webes erőforrások](https://docs.microsoft.com/en-us/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)
