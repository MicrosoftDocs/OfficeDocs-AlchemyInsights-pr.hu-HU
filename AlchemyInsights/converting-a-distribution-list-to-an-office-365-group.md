# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Terjesztési csoport átalakítása Microsoft 365-csoportká

A terjesztési csoportokat Microsoft 365-csoportokká konvertálhatja, ha megfelelnek bizonyos feltételeknek. A csoportok átalakításáról vagy az átalakítási hibák elhárításáról a Terjesztési listák [frissítése Microsoft 365-csoportokká](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists)az Outlookban című témakörben található.

A Microsoft 365-csoportra való frissítéshez futtathat egy parancsfájlt is, amely segít a terjesztési csoport [jogosultságának érvényességének igazolásában.](https://aka.ms/DLToM365Group)
