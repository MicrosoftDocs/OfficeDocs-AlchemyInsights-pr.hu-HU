# <a name="convert-a-distribution-group-to-an-microsoft-365-group"></a>Terjesztési csoport átalakítása Microsoft 365-csoporttá

A terjesztési csoportokat microsoft 365-ös csoportokká alakíthatja, ha megfelelnek bizonyos feltételeknek. Csoport konvertálásához vagy konvertálási problémák elhárításához olvassa el [a Terjesztési listák frissítése Microsoft 365 Csoportok ra az Outlook programban című témakört.](https://docs.microsoft.com/office365/admin/manage/upgrade-distribution-lists)
