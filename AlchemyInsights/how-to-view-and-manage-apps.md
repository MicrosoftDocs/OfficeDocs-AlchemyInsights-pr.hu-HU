# <a name="how-to-view-and-manage-apps"></a>Alkalmazások megtekintése és kezelése

Az alkalmazások a Microsoft 365 felügyeleti központban tekinthetők meg.

1. A felügyeleti központban lépjen a termékek **számlázása**  >  [Your products](https://go.microsoft.com/fwlink/p/?linkid=842054) lapra.
2. A **Products (termékek** ) lapon görgessen le az **alkalmazások** szakaszig.
3. Egy alkalmazás kezeléséhez jelölje ki a kezelni kívánt alkalmazást.
