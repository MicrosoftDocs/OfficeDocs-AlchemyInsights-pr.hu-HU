# <a name="migrating-to-microsoft-intune"></a>Áttérés a Microsoft Intune-ra

Ha a Intune-ra próbál át áttérni a Intune- [áttelepítési útmutatóba](https://docs.microsoft.com/intune/fundamentals/migration-guide).

Ha részletesebb útmutatást szeretne kapni a System Center Configuration Manager (és a SCCM Hybrid)-ról a Intune-ra való áttérésről, reivew a [hibrid áttelepítési dokumentációt](https://docs.microsoft.com/sccm/mdm/deploy-use/migrate-hybridmdm-to-intunesa).
