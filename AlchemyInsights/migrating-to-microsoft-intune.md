# <a name="migrating-to-microsoft-intune"></a>Áttelepítés a Microsoft Intune-ra

Ha az Intune-ra próbál áttérni, tekintse át az [Intune áttelepítési útmutatóját.](https://docs.microsoft.com/intune/fundamentals/migration-guide)

A System Center Configuration Managerből (és az SCCM Hybridből) az Intune-ba történő áttelepítéssel kapcsolatos további útmutatásért tadalkban tarthatja [a hibrid áttelepítési dokumentációt.](https://docs.microsoft.com/sccm/mdm/deploy-use/migrate-hybridmdm-to-intunesa)
