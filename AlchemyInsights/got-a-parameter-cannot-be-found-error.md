# <a name="migrating-email-from-imap-email-system-to-microsoft-365"></a>E-mailek áttelepítése IMAP-alapú rendszerből a Microsoft 365

Az e-mailek IMAP-alapú rendszerből való áttelepítéséhez kövesse az alábbi eljárást: [IMAP-áttelepítés a Microsoft 365 felügyeleti központjában](https://docs.microsoft.com/Exchange/mailbox-migration/migrating-imap-mailboxes/imap-migration-in-the-admin-center)
  
Sok nagy szervezetnek gondosan meg kell terveznie a felhasználói elvárásokat. Az [IMAP-postaládák Microsoft 365-be való áttelepítésével kapcsolatos tudnivalók](https://docs.microsoft.com/Exchange/mailbox-migration/migrating-imap-mailboxes/migrating-imap-mailboxes) a változtatások kezeléséhez szükségesek. Felhívjuk a figyelmét arra, hogy a G Suite (Google) áttelepítését [nem kell többé IMAP](https://docs.microsoft.com/Exchange/mailbox-migration/perform-g-suite-migration)-kapcsolaton keresztül végrehajtani.

Kérjük, hogy a [Microsoft 365 elfogadása a FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) lapon segítséget nyújtson az áttelepítéssel kapcsolatos segítségért, ha jogosult.
