# <a name="add-permissions"></a>Engedélyek hozzáadása

1. A rendszergazdai jóváhagyási végpont használatával konfigurálhatja egy alkalmazás statikusan kért [engedélyeinek listáját.](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application)
1. Két engedély áll rendelkezésre az alkalmazás-regisztrációk létrehozására, amelyek mindegyikének különböző viselkedése van. További információt az Azure Active Directory egyéni szerepköreinek alkalmazásregisztrációs engedélyeit [tartalmaz.](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions)
1. [Szerepkör engedélyei:](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions)az egyes szerepkörhöz adott Azure Active Directory-engedélyeket leíró táblák. Egyes szerepkörök további engedélyekkel is rendelkezniük kell az Azure Active Directoryn kívüli Microsoft-szolgáltatásokban.
