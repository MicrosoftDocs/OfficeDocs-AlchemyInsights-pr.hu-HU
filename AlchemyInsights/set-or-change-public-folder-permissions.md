# <a name="permissions-and-public-folders"></a>Engedélyek és nyilvános mappák

A nyilvános mappák engedélyeit az Outlook, az Exchange Admin Center (EAC) vagy a PowerShell segítségével módosíthatja.
  
- Az Outlookra vonatkozó utasításokért [kattintson ide](https://support.office.com/article/Set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005.aspx).
    
- Az Exchange felügyeleti központban a [jelen cikkben](https://technet.microsoft.com/library/jj651147%28v=exchg.150%29.aspx.aspx#Anchor_1) útmutatást találhat. 
    
- PowerShell esetén a [jelen cikk](https://technet.microsoft.com/library/bb124743%28v=exchg.160%29.aspx.aspx) útmutatást nyújt a Add-PublicFolderClientPermission parancsmagot használatáról. Ha útmutatásra van szüksége az Exchange Powershellhez való csatlakozáshoz, kattintson [ide](https://technet.microsoft.com/library/jj984289%28v=exchg.160%29.aspx.aspx).
    
Ha a **külső felhasználók nem tudnak e-maileket küldeni levelezési nyilvános mappába**, előfordulhat, hogy a nyilvános mappa hiányzik a külső e-mailek kézbesítéséhez szükséges engedélyekkel. Ezt az Outlook-utasításokkal vagy a PowerShell [-utasításokkal](https://technet.microsoft.com/library/aa997560%28v=exchg.150%29.aspx.aspx#Anchor_1)javíthatja [ki.](https://support.microsoft.com/help/2984402/-5.7.1-smtp-550-5.7.1-resolver.rst.authrequired-nondelivery-report-when-external-users-try-to-send-mail-to-mail-enabled-public-folders-in-office-365.aspx)
