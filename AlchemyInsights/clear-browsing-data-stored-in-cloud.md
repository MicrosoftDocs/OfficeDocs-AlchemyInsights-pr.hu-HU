# <a name="clear-the-browsing-data-stored-in-the-cloud-if-youve-turned-on-sync-or-personalization"></a>Törölje a felhőben tárolt böngészési adatokat (ha bekapcsolta a szinkronizálást vagy a személyre szabást)

A felhőbe mentett adatok megtekintéséhez, kezeléséhez vagy törléséhez használja az adatvédelmi irányítópultot. Az irányítópult használatával törölt adatok nem törlődnek az eszközről. Részletes információkért lásd: [Az adatok megtekintése az adatvédelmi irányítópulton.](https://support.microsoft.com/windows/view-your-data-on-the-privacy-dashboard-03d3e27f-1981-5ff4-ba1c-d6b1031ae433)
