# <a name="how-to-delete-a-folder-in-outlook-on-the-web"></a>A weben az Outlook mappa törlése

1. A mappalistán kattintson a jobb gombbal a törölni kívánt almappát.
2. Jelölje be a **mappa törlése**.
3. Kattintson az **OK gombra** megerősítéséhez.

Többet, olvassa el [a weben az Outlook üzenet mappákkal való munkára](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
