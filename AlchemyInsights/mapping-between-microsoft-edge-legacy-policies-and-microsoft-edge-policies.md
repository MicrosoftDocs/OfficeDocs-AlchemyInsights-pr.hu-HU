# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Tudnivalók a Microsoft Edge régi házirendek és a Microsoft Edge házirendek közötti megfeleltetésről

A Microsoft Edge régi verziójának (45-ös és korábbi verziók) csoportházirendei és a mobileszköz-kezelés (MDM) beállításai a Microsoft Edge 80-as verziójának megfelelő házirendjéhez vannak leképezve. További információ: A Microsoft Edge régi [verziójának megfeleltetése Microsoft Edge házirend-megfeleltetésre.](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge)
