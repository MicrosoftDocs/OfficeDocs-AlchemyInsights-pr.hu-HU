# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>PowerApps vagy a Microsoft Flow támogatási kérelmek küldése

Ha Ön egy **PowerApps** vagy **Microsoft áramlási** vevő ismertesse a jegy az új [Energiagazdálkodási platform Admin Center](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819)keresztül. Az új portál van optimalizálva, PowerApps és egyéb kiemelt platform termékek támogatás több közvetlen elérési útját tartalmazza.

[Microsoft Flow támogatási kérelem elküldése](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)
