# <a name="learn-how-microsoft-search-in-bing-helps-keep-your-information-secure"></a>További információ arról, hogyan segíti a Microsoft Search a Bingben az adatok biztonságát

A Microsoft Keresés funkció a Bingben:

- Csak a bejelentkezett felhasználók láthatják a **keresési** eredményeket.
- A Microsoft nem használja fel az Ön munkahelyi vagy szervezeti identitását a hirdetések megcéllára.
- A keresési rendszergazda típus szerint csak a keresések számát láthatja, például személy vagy fájltípus szerint, valamint a legfelső szintű keresések összesített listáját.
- Csak Ön láthatja a munkahelyi keresési **előzményeit.**
- Itt csak azokkal a munkahelyi fájlokkal és információkkal lehet megtalálni, amelyek meg vannak osztva az Ön számára, illetve amelyekhez hozzáférési jogosultsággal rendelkezik.
- Az adatai soha nem válnak elérhetővé más szervezetek, például a Microsoft, illetve a Bingen végzett nyilvános keresések számára.
