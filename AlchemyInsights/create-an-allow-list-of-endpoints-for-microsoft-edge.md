# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Végpontok engedélyezési listájának létrehozása a Microsoft Edge-hez

Ahhoz, hogy a Microsoft Edge funkcióit az Ön rendelkezésére bocsássák, csatlakoznia kell az internethez. Ha azt szeretné, hogy a böngésző a tűzfalon és más biztonsági mechanizmusokon keresztül kommunikáljon az interneten, a tartomány URL-címeinek listáját kell hozzáadnia a [végpontok engedélyezéséhez](https://go.microsoft.com/fwlink/?linkid=2135054).
