# <a name="using-custom-policy-in-intune"></a>Egyéni házirend használata az Intune-ban

Az Intune számos beépített beállítást tartalmaz az Android, iOS, macOS és Windows rendszerű eszközökhöz. Egyéni profilt is létrehozhat, majd ezeket az egyéni beállításokat a Intune segítségével telepítheti az eszközére. Az egyéni profilok olyan eszközbeállítások és funkciók hozzáadására szolgálnak, amelyek nem szerepelnek a Intune-ban.

[Egyéni beállítások az Android-eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-android)

[Egyéni beállítások az Android Enterprise-eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Egyéni beállítások az iOS-eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-ios)

[Egyéni beállítások a macOS eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-macos)

[Egyéni beállítások a Windows 10-es eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-windows-10)

[A Windows holografikus vállalati verziós eszközökhöz használható egyéni beállítások az Intune-ban](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
