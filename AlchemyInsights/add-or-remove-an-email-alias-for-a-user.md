# <a name="add-or-remove-an-email-address-for-a-user"></a>Hozzáadása vagy eltávolítása egy felhasználó e-mail címe

[Nyissa meg az első lépések](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions).
    
 **Megjegyzés**: a felhasználó egynél több e-mail címe is, de további *e-mail aliasok* csak e-mail fogadására használható. Ha egy aliasnévvel rendelkező e-mailt küldeni kell, hozzon létre egy [megosztott postafiókjából](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9). Így jelenik meg az alias levelek küldése és fogadása.
