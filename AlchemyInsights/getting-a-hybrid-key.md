# <a name="getting-an-exchange-hybrid-key"></a>Hibrid Exchange-kulcs beszerzése

Ha a helyszíni szervezetet a Microsoft 365-ben lévő Exchange Online-hoz szeretné csatlakoztatni, legalább egy Exchange-kiszolgálót be kell állítania a helyszíni szervezetben. Ezzel a kiszolgálóval hibrid telepítést hozhat létre, amely összeköti a helyszíni Exchange-szervezetet az Exchange Online-nal. A kiszolgáló hibrid kulcsának lekérnie, kövesse az [Exchange hibrid termékkulcs-terjesztési varázslójának lépéseit.](https://aka.ms/hybridkey)
