# <a name="using-office-product-keys"></a>Az Office termékkulcsainak használata

**Office termékkulcsok**

- Ha az Office-vásárláshoz termékkulcs érkezett, olvassa [el A Hol adhatja meg az Office-termékkulcsot.](https://support.office.com/article/Where-to-enter-your-Office-product-key-0a82e5ae-739e-4b92-a6f4-2ec780c185db)

- Az Office 365 Otthoni vagy Egyszemélyes verziós termékkulcsával kapcsolatban a [Termékkulcsok használata az Office-szal (Termékkulcsok használata az Office-sal) témakörben](https://support.office.com/article/using-product-keys-with-office-12a5763a-d45c-4685-8c95-a44500213759)talál segítséget.

**Office-előfizetés aktiválása** 
- Ha Microsoft 365-előfizetést vásárolt egy kiskereskedelmi üzletben, olvassa [el A Microsoft 365 Vállalati alkalmazások termékkulcsának megadása](https://docs.microsoft.com/microsoft-365/commerce/enter-your-product-key).

- Ha egy termékkulcs beváltásakor hibaüzenet jelenik meg, olvassa el [a Microsoft 365 vállalati verziós termékkulcssal kapcsolatos problémák című témakört?](https://docs.microsoft.com/microsoft-365/commerce/product-key-errors-and-solutions)
