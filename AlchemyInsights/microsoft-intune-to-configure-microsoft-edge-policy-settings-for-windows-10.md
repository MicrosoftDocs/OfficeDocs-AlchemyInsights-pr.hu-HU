# <a name="use-microsoft-intune-to-configure-microsoft-edge-policy-settings-for-windows-10"></a>A Microsoft Intune használata a Microsoft Edge windows 10-es házirendbeállításainak konfigurálhoz

A Microsoft Edge házirendeket és beállításokat úgy konfigurálhatja, hogy hozzáad egy eszközkonfigurációs profilt a Microsoft Intune-hoz. Ez egyenértékű a Microsoft Azure Active Directory csoportházirend-házirendek vagy helyi csoportházirend-objektum -beállítások felhasználói eszközökön történő konfigurálásával.

További információ: Profil létrehozása a beállítások kezeléséhez [a Windows 10 Microsoft Edge-ben.](https://go.microsoft.com/fwlink/?linkid=2133700)

Ez a cikk a Microsoft Edge 77-es és újabb verzióira vonatkozik.
