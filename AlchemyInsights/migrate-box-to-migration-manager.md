# <a name="migrate-box-to-microsoft-365-with-migration-manager"></a>A Box áttelepítése a Microsoft 365-be az Áttelepítéskezelővel

A Box-dokumentumokat, adatokat és felhasználókat a Microsoft 365-ben a OneDrive-ra, a SharePointra és a Teamsbe áttértve egyetlen helyen működhet együtt másokkal.

További információ: [A Box áttelepítése a Microsoft 365-be az Áttelepítéskezelővel.](https://docs.microsoft.com/sharepointmigration/mm-box-overview)
