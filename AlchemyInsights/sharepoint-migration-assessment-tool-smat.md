# <a name="sharepoint-migration-assessment-tool-smat"></a>SharePoint Migration Assessment Tool (SMAT)

A SharePoint áttelepítési felmérés eszköze (SMAT) egy egyszerű parancssori végrehajtható fájl, amely ellenőrzi a SharePoint-farm tartalmát, és segítséget nyújt a kiszolgáló SharePoint Online-ba való áttelepítésének a Microsoft 365-tel való áttelepítésében.

[A SharePoint Migration Assessment Tool (SMAT) áttekintése](https://docs.microsoft.com/sharepointmigration/overview-of-the-sharepoint-migration-assessment-tool)
