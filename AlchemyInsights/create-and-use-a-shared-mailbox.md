# <a name="create-and-use-a-shared-mailbox"></a>Megosztott postaláda létrehozása és használata

A megosztott postaláda lehetővé teszi, hogy egynél több személy küldjön és fogadjon e-mailt egy aliashoz, például sales@contoso.com, anélkül, hogy másik licencet rendelne.
  
- A kezdéshez [kövesse az alábbi lépéseket](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) a Microsoft 365 admin központban. 

Elolvashatja a [megosztott postaláda létrehozása](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) cikket, amely megmondja, hogy a megosztott postaládákról mit kell tudnia, beleértve a használatuk módját is.
