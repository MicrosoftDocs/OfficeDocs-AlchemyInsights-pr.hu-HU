# <a name="learn-about-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>További információ a Microsoft Edge korábbi házirendek és a Microsoft Edge-házirendek közötti megfeleltetésről

A Microsoft Edge Legacy (45-es és korábbi verziói), valamint a mobileszköz-kezelés (MDM) beállításainak megfeleltetése a Microsoft Edge-es verzió 80-es verziójának megfelelő házirendjeire. További információt a [Microsoft Edge Legacy to Microsoft Edge Policy megfeleltetés](https://go.microsoft.com/fwlink/?linkid=2141665)című témakörben találhat.
