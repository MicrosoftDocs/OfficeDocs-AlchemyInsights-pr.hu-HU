# <a name="troubleshooting-issues-with-calendar-sharing"></a>A naptármegosztással kapcsolatos problémák elhárítása

- Az engedélyekkel kapcsolatos problémák diagnosztizálásához használja az [Exchange Remote Connectivity Analyzer Szabad elfoglaltságot](https://testconnectivity.microsoft.com/Default.aspx?testId=freeBusy), amely nagyon specifikus hibaüzeneteket ad.
- A [Microsoft Microsoft 365 támogatási és helyreállítási segédjével](https://diagnostics.office.com/) számos más naptárproblémát is diagnosztizálhat. 
- Ha a felhasználó rendelkezik az Asztali Outlook programmal, az [Outlook naptárellenőrző eszköze](https://www.microsoft.com/download/details.aspx?id=28786) megoldhatja a lehetséges problémákat.

További útmutatók:

- [Outlook-naptár megosztása másokkal](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)
- [Naptár megosztása a Webes Outlookban](https://support.office.com/article/7ecef8ae-139c-40d9-bae2-a23977ee58d5)
