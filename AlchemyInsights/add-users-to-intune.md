# <a name="add-users-to-intune"></a>Felhasználók hozzáadása az Intune szolgáltatáshoz

A [Microsoft 365 Admin Center](https://admin.microsoft.com/) vagy az [Intune Portal](https://portal.azure.com/#blade/Microsoft_Intune_DeviceSettings/ExtensionLandingBlade/overview)használatával manuálisan is hozzáadhat felhasználókat az Intune előfizetéshez.

A rendszergazda módosíthatja a felhasználói fiókokat az Intune licencek hozzárendeléséhez. Licenceket rendelhet a Microsoft 365 admin vagy az Intune portálhoz.

A Microsoft 365 Admin Center használatáról további információt a [felhasználók hozzáadása egyenként vagy ömlesztve a microsoft 365 admin Centerből](https://support.office.com/article/Add-users-individually-or-in-bulk-to-Office-365-Admin-Help-1970f7d6-03b5-442f-b385-5880b9c256ec)című témakörben talál.

Az Intune portál használatáról további információt a [felhasználók hozzáadása és a felügyeleti engedélyek megadása az Intune](https://docs.microsoft.com/intune/fundamentals/users-add)szolgáltatáshoz című témakörben talál.
