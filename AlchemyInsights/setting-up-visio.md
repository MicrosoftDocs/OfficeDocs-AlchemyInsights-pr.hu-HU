# <a name="setting-up-visio"></a>A Visio beállítása

Úgy tűnik, segítségre van szüksége a Visio Microsoft 365-előfizetéssel való beállításához.
  
Figyelmeztetés Mac-felhasználóknak: A Visio-termék jelenleg nem érhető el MacOS rendszeren. Ha van számítógépe, az alábbi lépések segítségével telepítheti a szoftvert.
  
A beállítás elvégzéséhez kövesse az [ebben a témakörben](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) található lépéseket.
