# <a name="public-folder-access-from-mobile-devices"></a>Nyilvános mappa elérése mobileszközökről

A nyilvános mappák nem érhetők el mobileszközökről.
  
Ha a nyilvános mappákhoz hasonló funkciókkal szeretne rendelkezni, és mobileszközökről szeretné elérni, olvassa el a Jobb alternatívát a [Microsoft 365-csoportok ról szóló témakört.](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)
