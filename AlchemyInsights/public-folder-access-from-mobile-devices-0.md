# <a name="public-folder-access-from-mobile-devices"></a>Nyilvános mappa hozzáférési mobileszközökről

A nyilvános mappák a hordozható eszközök nem érhetők el.
  
Ha azt szeretné, a nyilvános mappák hasonló funkciókkal rendelkezik, és elérhető mobil eszközök, [Office 365 csoportokkal kapcsolatos további tudnivalók](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) talál jobb megoldás.
