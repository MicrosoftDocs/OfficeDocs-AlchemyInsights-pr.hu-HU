# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Az Outlook alapértelmezés szerint blokkolja a külső tartalmat az S/MIME-ben

További biztonsági intézkedéseket vezetünk be a Windows Outlookban azáltal, hogy alapértelmezés szerint letiltjuk a külső tartalmat az S/MIME titkosított és aláírt üzenetekben, és megakadályozza a felülírásokat üzenetenként.

A részleteket lásd: [Outlook for Windows blocks external content in S/MIME messages](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e).
