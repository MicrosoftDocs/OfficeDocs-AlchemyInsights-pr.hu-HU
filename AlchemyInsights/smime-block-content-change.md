# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Az Outlook ekkor alapértelmezés szerint letiltja a külső tartalmakat az S/MIME-ben

A Windows Outlookban további biztonsági intézkedéseket vezetünk be, amelyek alapértelmezés szerint blokkolják az S/MIME-alapú titkosított és aláírt üzenetekben lévő külső tartalmakat, és így megakadályozzák a felülírást egy üzenet alapján.

A teljes részletekért olvassa el a [Windows Outlook blokkolja a külső tartalmat az S/MIME-üzenetekben](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e)című témakört.
