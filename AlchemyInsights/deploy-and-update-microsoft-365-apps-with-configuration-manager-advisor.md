# <a name="deploy-and-update-microsoft-365-apps-with-configuration-manager-advisor"></a>A Microsoft 365-alkalmazások telepítése és frissítése a Configuration Manager Advisor segítségével

A Configuration Managert használó szervezetek esetében használhatja a [microsoft 365-alkalmazások telepítése és frissítése a Configuration Manager Advisor](https://admin.microsoft.com/adminportal/home#/oppinstall) segítségével olyan parancsfájl generálását, amely a FastTrack-mérnökök által ajánlott ajánlott eljárások segítségével automatikusan beállítja a Microsoft 365-alkalmazásokat.

Ezzel az útmutatóval felépítheti a központi telepítési csoportokat, testre szabhatja az Office-alkalmazásokat és-szolgáltatásokat, beállíthatja a dinamikus vagy a sovány telepítést, majd a parancsfájl futtatásával létrehozhatja az alkalmazásokat, az automatikus telepítési szabályokat és az eszközök gyűjteményét, amelyekre a telepítést meg kell céloznia.
