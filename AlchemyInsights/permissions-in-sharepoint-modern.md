# <a name="permissions-in-the-sharepoint-modern-experience"></a>Engedélyek a SharePoint modern felületén

A SharePoint-engedélyeket hagyományosan a webhely engedélyeinek csoportjai (tulajdonosok, tagok, látogatók stb.) között kezelték. A SharePointban a Microsoft 365-ben ez a funkció bizonyos típusú webhelyek esetén is igaz, de további lehetőségek is elérhetők.  

További információért olvassa el [a SharePoint modern felületének megosztása és engedélyei](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions)című témakört.
