# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Éva segítségével azonosíthatja és megoldhatja a hitelesítési problémák az Outlook

Erősen ajánlott az [Outlook leáll kéri a jelszó](https://aka.ms/SaRA-OutlookPwdPrompt) diagnosztikai az érintett számítógépen problémák Ha az Outlook folyamatosan jelszót kér. A [Éva](https://diagnostics.office.com/#/) diagnosztikai automatizált ellenőrzések és az függvény segítségével bármely észlelt problémák lehetséges megoldásokat.
