# <a name="desktop-app-for-sharepoint"></a>Asztali SharePoint-alkalmazás

Szervezete bekerül a SharePoint asztali alkalmazás privát előzetes verziójában.

A Microsoft jelenleg nem támogatja a SharePointhoz készült asztali alkalmazást (magánjellegű előzetes verzió). Kérjük, hogy a Microsoft privát előzetes verziójának partnerei segítséget kapjanak a SharePoint asztali alkalmazásával kapcsolatban.
