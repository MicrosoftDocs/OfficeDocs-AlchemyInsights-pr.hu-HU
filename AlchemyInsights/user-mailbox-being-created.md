# <a name="your-users-mailbox-is-being-created"></a>A felhasználói postafiók létrehozása folyamatban van

Azt is igénybe vehet egy óra előtt egy új postaládát használatra készen áll.
  
Ha a felhasználó postafiókja van már "előkészítése" több mint egy óra, próbálja törlése és hozza létre újra a felhasználó, vagy egy támogatási mérnök beszélgetni "tudassa velünk hívja meg" lehetőség használatával.
