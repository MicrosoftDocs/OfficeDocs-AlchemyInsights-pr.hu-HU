# <a name="your-users-mailbox-is-being-created"></a>A felhasználó postaládája létrehozása folyamatban van

Az új postaláda használatra kész beállítása akár egy órát is igénybe vehet.
  
Ha a felhasználó postaládája több mint egy órája "készül", megpróbálhatja a felhasználó törlését és újbóli létrehozását, vagy használhatja a "Szólítsuk Önt" lehetőséget, hogy beszéljen egy támogatási szakemberrel.
