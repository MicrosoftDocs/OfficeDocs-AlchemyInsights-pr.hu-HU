# <a name="assign-or-remove-users-licenses"></a>Felhasználói licencek hozzárendelése vagy eltávolítása

Rendszergazdaként szükség szerint módosíthatja a felhasználók licenceit. Az alábbi cikkek bemutatják, hogyan kezelheti a felhasználók licenceit, beleértve a következőket:
  
- [Licencek hozzárendelése felhasználókhoz a Microsoft 365 vállalati verzióban](https://docs.microsoft.com//office365/admin/subscriptions-and-billing/assign-licenses-to-users)

- [Licencek visszavonása a Microsoft 365 vállalati verziófelhasználóitól](https://docs.microsoft.com//office365/admin/subscriptions-and-billing/remove-licenses-from-users)

Nem válaszolt a kérdésére? nem gond! Használja a "Segítségre van szüksége" gombot, hogy beszéljen egy támogatási szakemberrel.
