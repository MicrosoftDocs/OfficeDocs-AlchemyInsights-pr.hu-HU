# <a name="set-up-microsoft-information-protection"></a>A Microsoft Information Protection beállítása

Áttekintést kaphat az adatvédelemi stratégiával kapcsolatos lehetőségekről, így biztos lehet benne, hogy bizalmas információkat védett. Használjon olyan négy szakaszos életciklusos megközelítést, amelyben bizalmas információkat talál, osztályozhat, véd és figyel. A "Microsoft Information Protection Setup" útmutató útmutatást ad az egyes szakaszok kitöltéséhez.
