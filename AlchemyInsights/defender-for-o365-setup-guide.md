# <a name="microsoft-defender-for-office-365-setup-guide"></a>Telepítési útmutató az Office 365-hez való Microsoft Defenderhez

Az [Office 365-nek](https://go.microsoft.com/fwlink/?linkid=2146614) megfelelő Microsoft Defender beállítási útmutató segít megvédeni szervezetét az e-mail-üzeneteken, hivatkozásokon és külső együttműködési eszközöken keresztül esetleg felmerülő rosszindulatú fenyegetésekkel szemben. Ez az útmutató a szervezet igényeinek megfelelő Microsoft Defender-terv előkészítéséhez és azonosításához szükséges erőforrásokat és információkat tartalmaz.
