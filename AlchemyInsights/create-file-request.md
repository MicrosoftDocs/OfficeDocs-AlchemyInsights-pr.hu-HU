# <a name="how-to-create-a-file-request"></a>Fájlkérés létrehozása

A OneDrive fájlkérési funkciójában kiválaszthatja azt a mappát, ahová mások feltöltik a fájlokat az Ön által nekik küldött hivatkozással. Azok a felhasználók, akiktől fájlokat kér, nem látják a mappa tartalmát – csak fájlokat tölthetnek fel a mappába.

**Megjegyzés:** A fájlkérések csak akkor működnek, ha egy rendszergazdának engedélyezi [a](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off) Bárki hivatkozásokat a OneDrive-ban.

- További információ: Fájlkérés [létrehozása.](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af)
