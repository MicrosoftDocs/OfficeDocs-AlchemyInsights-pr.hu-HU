# <a name="transfer-ownership-of-a-microsoft-form"></a>A Microsoft formában tulajdonjogának átruházása

Áthelyezheti egy felmérés, kvíz vagy lekérdezési a Microsoft Forms csoporthoz, a csoport minden tagja számára az adott űrlap tulajdonosok válik. Is átvihetők űrlap tulajdonjogot egy másik felhasználónak, ha a korábbi tulajdonos elhagyta a szervezet.

További információt talál [az űrlap tulajdonjogának átruházása](https://support.office.com/article/Transfer-ownership-of-a-form-921a6361-a4e5-44ea-bce9-c4ed63aa54b4).
