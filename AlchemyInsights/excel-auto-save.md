# <a name="what-is-autosave"></a>Mi az automatikus mentés?

Az automatikus mentés funkció a Microsoft 365-előfizetőknek készült Excel, Word és PowerPoint alkalmazásban elérhető funkció, amellyel néhány másodpercenként automatikusan menti a fájlt. 

A gyakori kérdések a [Mi az automatikus mentés?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5) című témakörben olvashatók.
