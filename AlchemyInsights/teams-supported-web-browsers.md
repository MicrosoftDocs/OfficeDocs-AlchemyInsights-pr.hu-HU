# <a name="teams-supported-web-browsers"></a>Csapatok által támogatott webböngészők

A Teams teljes mértékben támogatja a Microsoft [Teams korlátozásai és specifikációi](https://docs.microsoft.com/en-us/microsoftteams/limits-specifications-teams#browsers)című cikkben felsorolt internetes böngészőket, a hívások és értekezletek esetében is ismert kivételekkel.
