# <a name="limit-sharing-in-microsoft-365"></a>Megosztás korlátozása a Microsoft 365-ben

Bár nem tilthatja le teljesen a belső megosztást, és nem távolíthatja el a Megosztás gombot a webhelyekről, számos módon korlátozhatja a Microsoft 365-ben a megosztást a szervezet igényeinek megfelelően. További információt [a Megosztás korlátozása a Microsoft 365-ben](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)című témakörben talál.
