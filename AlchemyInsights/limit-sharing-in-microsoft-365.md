# <a name="limit-sharing-in-microsoft-365"></a>Megosztás korlátozása a Microsoft 365-ban

Noha nem lehet teljes mértékben letiltani a belső megosztást, vagy el szeretné távolítani a webhelyek megosztás gombját, többféleképpen is korlátozhatja a megosztást a Microsoft 365-ban, hogy megfeleljen a szervezet igényeinek. További információt a [megosztás korlátozása a Microsoft 365-ban](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)című témakörben találhat.
