# <a name="microsoft-365-language-settings"></a>A Microsoft 365 nyelvi beállításai

Határozza meg, hogy szeretné-e módosítani a felhasználói vagy a teljes bérlői fiók megjelenítési nyelvét és egyéb beállításait.

- A felhasználók számára A megjelenítési nyelv és az időzóna módosítása a [Microsoft 365 Vállalati verzióban](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b)az összes alkalmazás esetében.
- A bérlői fiókhoz  [lásd: A Microsoft 365](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region)nyelvi és területi beállításainak megadása.
