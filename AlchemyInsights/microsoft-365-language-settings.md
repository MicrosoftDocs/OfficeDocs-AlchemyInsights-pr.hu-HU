# <a name="microsoft-365-language-settings"></a>A Microsoft 365 nyelvi beállításai

Állapítsa meg, hogy szeretné-e módosítani a felhasználói felület és a teljes bérlő nyelvének megjelenítési nyelvét és egyéb beállításait.

- A felhasználók számára a felhasználói [felület nyelvének és időzónájának módosítása a Microsoft 365 vállalati verziójában minden alkalmazásban](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b).
- A bérlői  [webhelyhez a Microsoft 365 nyelvének és területi beállításainak megadása](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region)című témakörben tájékozódhat.
