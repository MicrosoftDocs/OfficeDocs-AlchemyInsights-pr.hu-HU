# <a name="increase-onedrive-storage"></a>A OneDrive-tárterület növelése

Globális vagy SharePoint-rendszergazdaként a Microsoft 365-ban megadhatja, hogy egy adott felhasználó OneDrive tárterülete közvetlenül a Microsoft felügyeleti központból történjen.  

További információt az [adott felhasználó OneDrive-tárterületének módosítása](https://docs.microsoft.com/onedrive/change-user-storage)című témakörben találhat.
