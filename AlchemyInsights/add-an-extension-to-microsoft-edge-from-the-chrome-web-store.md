# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Bővítmény hozzáadása a Microsoft Edge-hez a Chrome Web áruházból

Ha bővítményt szeretne hozzáadni a Microsoft Edge-hez a Chrome Web áruházból, végezze el az alábbi lépéseket:

1. Nyissa meg a Chrome Web áruház webhelyet.
2. A lap tetején lévő szalagcímen válassza a **bővítmények engedélyezése más áruházból** lehetőséget.
3. Jelölje ki a hozzáadni kívánt bővítményt, és kattintson a **Hozzáadás a Chrome-hoz** gombra.
4. A megerősítést kérő párbeszédpanelen figyelmesen nézze át a bővítmény lehetséges módosításait, majd kattintson a **kiterjesztés hozzáadása** gombra.
A végleges párbeszédpanel megerősíti a bővítmény hozzáadását.
