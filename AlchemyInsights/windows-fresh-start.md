# <a name="windows-fresh-start"></a>Windows új kezdet

A Windows Új indítás eszköz művelet alternatívát kínál az Eszköztörlés és az Eszközkivonás művelethez, valamint az eszközregisztrációs állapot megőrzésének lehetőségét a felhasználói kezdőmappa-alkalmazásokkal és beállításokkal együtt.

További információ: [Az Új kezdet használata a Windows 10-es eszközök alaphelyzetbe állításához az Intune-nal.](https://docs.microsoft.com/intune/device-fresh-start)
