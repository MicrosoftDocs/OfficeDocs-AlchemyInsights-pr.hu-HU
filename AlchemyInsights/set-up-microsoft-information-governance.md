# <a name="set-up-microsoft-information-governance"></a>A Microsoft Information-irányítás beállítása

A [Microsoft adatkezelési beállítási útmutatója](https://admin.microsoft.com/AdminPortal/Home#/modernonboarding/migsetupguide) segítségével megadhatja a szervezeti stratégia beállításához és kezeléséhez szükséges információkat, így biztosíthatja, hogy az adatok a megadott életciklus-irányelvek szerint legyenek osztályozva és kezelhetők.

Ezzel az útmutatóval megtudhatja, hogy miként hozhat létre, alkalmazhat automatikusan vagy közzétehet címkék, címke-és adatmegőrzési házirendeket a szervezet újrahasználható tartalmi és megfelelőségi rekordjain. A CSV-fájlok importálására vonatkozó információkat a tömeges esetekben vagy az egyes dokumentumokra való manuális alkalmazáskor is megtalálhatja.
