# <a name="how-to-select-all-in-outlookcom"></a>"Az egészet kijelölni" az Outlook.com módjáról

Választhat az összes látható az összes üzenet esetén. Ez az üzenet nem jelenik meg a véletlen műveletek elkerülése érdekében.

**CTRL + kattintás** kijelöli az egyes üzenetek több.

**SHIFT + kattintás** kijelöli az összes üzenet között az első és utolsó üzenetet választja.
