# <a name="sync-warning-last-synced-more-than-3-days-ago"></a>Szinkronizálási figyelmeztetés: utolsó szinkronizálása több mint 3 nappal ezelőtt

További hibaelhárításért lásd a [címtár-szinkronizálási problémák megoldása](https://docs.microsoft.com/office365/enterprise/fix-problems-with-directory-synchronization)című témakört.

A szinkronizálás állapotának ellenőrzéséhez kövesse az Office Admin portálon a címtár- [szinkronizálási állapot megtekintése](https://docs.microsoft.com/office365/enterprise/view-directory-synchronization-status)című témakör útmutatását.
