# <a name="issues-with-validating-tokens"></a>A jogkivonatok érvényességének ával kapcsolatos problémák

A tokenek érvényesítésével kapcsolatos hibák elhárításához lásd:

- [Tokenek érvényességének igazolása](https://docs.microsoft.com/azure/active-directory/develop/access-tokens#validating-tokens) – A hozzáférési jogkivonatok érvényességének igazolása.

- [Az azonosítótoken ellenőrzése](https://docs.microsoft.com/azure/active-directory/develop/v2-protocols-oidc#validate-the-id-token) – Az azonosítótokenek érvényesítésével kapcsolatos információkért.
