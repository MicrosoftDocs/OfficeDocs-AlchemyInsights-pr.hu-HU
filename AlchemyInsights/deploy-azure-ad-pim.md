# <a name="deploy-azure-ad-privileged-identity-management-pim"></a>Az Azure AD privilegizált identitás-kezelés (PIM) telepítése

Ha meg szeretné tudni, hogy miként tervezheti meg az Azure Active Directory-szervezetben az "privilegizált Identitáskezelés" (PIM) alkalmazást, olvassa el az [Azure ad kiváltságos Identitáskezelő (PIM) telepítése](https://go.microsoft.com/fwlink/?linkid=2132095)című témakört.
