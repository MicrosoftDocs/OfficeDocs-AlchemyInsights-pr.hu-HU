# <a name="upload-process-overview-in-microsoft-stream"></a>Feltöltési folyamat áttekintése a Microsoft streamben

A videók feltöltésekor a Microsoft stream automatikusan több bitrátába kódolja azt, így a videót adaptív bitráta folyamatos átvitelsel is továbbíthatja. A Microsoft stream a nézők elérhető sávszélessége alapján optimalizálja a lejátszási bitrátát és a videoklip minőségét.
