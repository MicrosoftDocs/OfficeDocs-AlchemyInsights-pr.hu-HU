# <a name="enable-or-disable-forms-pro"></a>A Forms Pro engedélyezése vagy letiltása

Ha a Microsoft Forms Pro alkalmazást próbálja engedélyezni, az a [Dynamics 365 vállalati](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant) licenc részeként jelenik meg. Másik lehetőségként a rendszergazda [microsoft Forms Pro licencet](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant)is vásárolhat. Microsoft [Forms Pro licencet](https://docs.microsoft.com/forms-pro/purchase#disable-microsoft-forms-pro-for-a-user-1) is eltávolíthat a felhasználótól.
