# <a name="create-a-folder"></a>Hozzon létre egy mappát

1. A bal oldali ablaktáblán jelölje ki az **Új mappa létrehozása**![létrehozása az új mappa gombra](media/d8e28612-fbdb-4d28-a4d0-14f7834cfd97.png) **mappák**mellett. 
    
2. Az új mappa mezőbe írja be a mappa nevét, és nyomja le az ENTER billentyűt.
    
További információért lásd [az Outlook.com mappa létrehozása](https://go.microsoft.com/fwlink/p/?linkid=873114).
