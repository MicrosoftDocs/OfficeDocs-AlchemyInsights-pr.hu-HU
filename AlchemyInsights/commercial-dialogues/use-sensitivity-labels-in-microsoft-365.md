# <a name="use-sensitivity-labels-in-microsoft-365"></a>Bizalmasság-címkék használata a Microsoft 365-ben

Bizalmasság-címkék használatával:
- A feliratos tartalmakra védelmi beállításokat, például titkosítást vagy vízjeleket is alkalmazva.
- Megvédheti az Office-appok tartalmát a különböző platformokon és eszközökön.
- Külső felek alkalmazásában és szolgáltatásaiban található tartalmak védelme a Microsoft Cloud App Security használatával.
- Bizalmasság-címkék kiterjesztése külső alkalmazásokra és szolgáltatásokra.
- A tartalom besorolása védelmi beállítások használata nélkül.

További információ: Bizalmasság-címkék [megismerása.](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels)
