# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Windows 10-es eszköz beállítása a Microsoft Defender Komplex veszélyforrások elleni védelemhez (MDATP)

Az MDATP az alábbi telepítési eszközöket és módszereket támogatja a Windows 10-ben:

- Csoportházirend
- Microsoft Endpoint Configuration Manager
- Mobileszköz-kezelés, beleértve a Microsoft Intune-t
- Helyi parancsfájlok

További információért olvassa el a Windows 10-es eszközök be- és [telepítőeszközeit és metódusait.](https://go.microsoft.com/fwlink/?linkid=2143460)
