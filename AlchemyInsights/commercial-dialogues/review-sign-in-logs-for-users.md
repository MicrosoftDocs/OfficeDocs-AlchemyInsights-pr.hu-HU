# <a name="review-sign-in-logs-for-users"></a>Felhasználók bejelentkezési naplóinak áttekintése

Ehhez tegye a következőt:

1. Az Azure Active Directory **felügyeleti**  >  **[központjának megnyitásához.](https://go.microsoft.com/fwlink/p/?linkid=2067268)**
1. Válassza **az Azure Active**  >  **Directory-bejelentkezések lehetőséget.**
1. Ha még nincs Azure AD P1- vagy P2-előfizetése, a rendszer kérni fogja, hogy indítson egy ingyenes prémium próbaverziót. Ha szeretné, kattintson **a Start a free Premium trial (Ingyenes** prémium próbaverzió) gombra, majd az Azure AD Premium P2 párbeszédpanelen válassza az Ingyenes **próbaaktivált**  >  **lehetőséget.** Az ingyenes próbaverzió 30 napig tart.
