# <a name="workflows-in-sharepoint"></a>A SharePoint munkafolyamatok

<p>SharePoint-munkafolyamat olyan előre programozott mini-alkalmazások, amelyek modernizálása és számos üzleti folyamatok automatizálásához. A munkafolyamatok terjedhet gyűjtő aláírások, visszajelzést vagy a terv vagy a dokumentum, jóváhagyások rutin eljárás aktuális állapotának nyomon követése.</p> <p><strong>Az első lépések:</strong> Ha meg&rsquo;re csak Bevezetés a SharePoint-munkafolyamatok, lásd: <a href="https://support.office.com/en-us/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3">Bevezetés a SharePoint-munkafolyamatok</a> további információért.</p> <p><strong>Microsoft áram:</strong> Fontolja meg a klasszikus SharePoint munkafolyamat-megoldások Microsoft Flow helyett. További információért lásd: <a href="https://support.office.com/en-us/article/create-a-flow-for-a-list-or-library-in-sharepoint-online-or-onedrive-for-business-a9c3e03b-0654-46af-a254-20252e580d01">Create egy lista vagy tár a SharePoint Online vagy a OneDrive folyamata</a>.</p> <p><strong>Hibák elhárításakor munkafolyamat:</strong> Hibakeresés a munkafolyamat hasonlít az előző verziók. További információért lásd: <a href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/debugging-sharepoint-server-workflows">Hibakeresés SharePoint-munkafolyamatok</a>. További információ a hibaelhárítás, lásd: <a title="SharePoint Server – problémamegoldás munkafolyamat érvényesség-ellenőrzési hibák" href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio">A SharePoint Server munkafolyamat érvényesség-ellenőrzési hibák elhárítása</a>.&nbsp;</p>
