# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 a hozzáférés megtagadva, a bérlő túllépte a küszöböt

Az üzenet akkor jelenik meg, ha túl sok spam vagy tömeges levelek küldésével Office 365 szervezetben. A blokk eltávolításához kell megérteni és magyarázza el az OK, a támogatás munkatársával, valamint javítsa ki a hiba okát.
