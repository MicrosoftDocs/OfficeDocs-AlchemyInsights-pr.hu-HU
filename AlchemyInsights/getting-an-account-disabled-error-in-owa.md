# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Hiba az Outlook programban a weben egy fiók le van tiltva?

Ha **a fiók le van tiltva** a hibát kapunk a `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` próbál megnyitni az Outlook a weben (korábbi nevén az Outlook Web App), amikor a szervezet admin lehet, hogy letiltotta a hozzáférést az Outlookhoz a weben. A rendszergazda [engedélyezheti vagy letilthatja az Outlook Web App egy postaláda](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)témakör utasításait követve is kijavíthatja a hibát.
