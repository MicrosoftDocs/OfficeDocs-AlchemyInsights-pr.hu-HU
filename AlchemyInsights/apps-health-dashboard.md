# <a name="microsoft-365-apps-health-dashboard-in-the-microsoft-365-apps-admin-center"></a>A Microsoft 365-alkalmazások állapot-irányítópultja a Microsoft 365 Alkalmazások felügyeleti központjában

A Microsoft 365-alkalmazások felügyeleti központjának Microsoft 365-alkalmazások állapot-irányítópultja segítséget nyújt az Office-appok naprakészentartásában azáltal, hogy betekintést nyújt az ügyféleszközökön való teljesítményükbe, és útmutatást nyújt a problémák optimalizálásához és elhárításához. További információt A [Microsoft Apps](https://docs.microsoft.com/deployoffice/admincenter/overview) Felügyeleti központ és a Microsoft [365-appok állapota](https://docs.microsoft.com/deployoffice/admincenter/microsoft-365-apps-health)– áttekintés című témakörben talál.
