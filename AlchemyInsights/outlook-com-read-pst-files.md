# <a name="pst-data-files-in-outlookcom"></a>PST adatfájlok Outlook.com

Az Outlook program nem importálhat. PST adatfájlt a Outlook.com fiókjába egy böngésző segítségével.

A importálása. PST-adatfájlt, az asztali Outlook alkalmazást kell használnia. További információ: [E-mailek, névjegyek és naptár importálása Outlook .pst alkalmazásból.](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac?wt.mc_id=Office_Outlook_com_Alchemy)
