# <a name="create-custom-content-types"></a>Egyéni tartalomtípusok létrehozása

1. Jelölje ki a fontos rovatoksávon a dolgozni kívánt listát, vagy kattintson a **Beállítások** (fogaskerék ikon) gombra, majd a **Webhely beállításai parancsra.**
2. Kattintson **a webhelytartalom-típusokra** a **WebDesigner-gyűjtemények csoportban.**
3. A Webhely tartalomtípusai lapon kattintson a Létrehozás gombra.
4. Az Új webhelytartalom-típus lapon adja meg az egyéni tartalomtípus adatait.

További információt az Egyéni tartalomtípusok ( [Egyéni tartalomtípusok](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991)) területen talál.
