# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>A Google Chrome-házirendek és a Microsoft Edge-házirendek közötti megfeleltetés ismertetése

A Google Chrome-házirendek a böngészőben a 80-es verzióban támogatott megfelelő Microsoft Edge-házirendekhez vannak rendelve. További információt a [Google Chrome a Microsoft Edge-házirendek megfeleltetése](https://go.microsoft.com/fwlink/?linkid=2141933)című témakörben talál.
