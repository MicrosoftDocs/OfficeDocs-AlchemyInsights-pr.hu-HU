# <a name="form-header---web-resource"></a>Űrlapfejléc – webes erőforrás

Az űrlap webes erőforrásait hozzáadhatja vagy szerkesztheti, hogy az űrlap vonzóbbá vagy hasznosabbá tegye a felhasználók számára. Először győződjön meg arról, hogy rendelkezik a rendszergazda i vagy a Rendszertestreszabó biztonsági szerepkörrel vagy azzal egyenértékű engedélyekkel.  

**Megjegyzés:** Webes erőforrás nem szerepelhet űrlapfejben vagy élőlábban.

További információt a [Webes erőforrás létrehozása és szerkesztése űrlapon című témakörben talál.](https://docs.microsoft.com/dynamics365/customer-engagement/customize/create-edit-web-resources#create-and-edit-a-web-resource-on-a-form)
