# <a name="activating-windows-10-subscriptions"></a>A Windows 10 rendszer előfizetések aktiválása

A Microsoft 365 vállalati prémium verzió, az Office 365 oktatási verzió, a Microsoft 365 és a nagyvállalati verzió digitális licenceket biztosít a Windows 10 rendszerhez az előfizetés részeként. Az előfizetés aktiválásához a Windows 10 Pro (1703-as vagy újabb verzió) alkalmazásnak már aktiválva kell lennie az eszközön. A Windows 10-előfizetés aktiválási követelményeinek teljes listáját a [Windows 10 előfizetés aktiválási követelményei](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements)című témakörben tekintheti meg.
