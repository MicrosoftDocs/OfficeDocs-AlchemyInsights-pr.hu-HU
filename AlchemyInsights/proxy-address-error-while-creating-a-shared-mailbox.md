# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>A proxycímek hibája a megosztott postaláda létrehozásakor

Ha megpróbált létrehozni egy megosztott postaládát, és a következő hibaüzenet jelenik meg: "a proxycímek" SMTP:alias@domain.com "már használatban van"... "a választott e-mail-címet már egy másik postaláda veszi fel a szervezetében.
  
Meg kell keresnie azt a felhasználót, csoportot vagy megosztott postaládát, amely e-mail-címmel rendelkezik, és törölheti vagy módosíthatja az e-mail-címét. Ezután létrehozhat egy megosztott postaládát a felszabadított e-mail-címmel. Keresse meg a Kezdőlap keresés hivatkozását.
  
Ha nem szeretné törölni a meglévő e-mail-címet, válasszon egy új e-mail-címet a megosztott postaládához.
