# <a name="install-project-professional"></a>A Project Professional telepítése

A Project [telepítése](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) című témakörből megtudhatja, hogy miként telepítheti a Project Online előfizetéses verzióját, valamint a Project 2019, 2016 vagy 2013 nem előfizetéses verzióját. 

A Project 2010 vagy a 2007 esetén kövesse az [office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d)   vagy az [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657)című témakör lépéseit. 

A projektet különálló alkalmazásként értékesítjük, így ha még nem tette meg, akkor [vásároljon vagy próbálkozzon a Project](https://products.office.com/project)alkalmazással.
