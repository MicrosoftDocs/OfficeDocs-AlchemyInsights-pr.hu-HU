# <a name="restore-the-sharepoint-root-site"></a>A SharePoint-gyökérwebhely visszaállítása

Állítsa vissza a törölt gyökérwebhelyet az új SharePoint Felügyeleti központból. A visszaállítás után cserélje le a webhelyet.

1. Ugrás a [Törölt webhelyek lapra.](https://admin.microsoft.com/sharepoint?page=recycleBin&modern=true) 
2. Jelölje ki a visszaállítani kívánt gyökérwebhelyet.
3. Kattintson a **Visszaállítás** elemre.
4. [Cserélje le a gyökérwebhelyet.](https://docs.microsoft.com/sharepoint/troubleshoot/sites/url-that-resides-under-root-site-collection-is-broken)

További információt a [gyökérwebhely modernizálása lapon található.](https://docs.microsoft.com/sharepoint/modern-root-site)
