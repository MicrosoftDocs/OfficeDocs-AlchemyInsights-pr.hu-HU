# <a name="in-app-support-in-outlook-for-windows"></a>Alkalmazáson belüli támogatás a Windows Outlookban

Bérlői rendszergazdaként alkalmazáson belüli támogatást kaphat a Windows Outlook kal kapcsolatos problémákhoz. Az Outlookban **Help**válassza a  >  **Súgó ügyfélszolgálatát**. Adja meg a kérdését, és az Outlook for Windows támogatási csoportjának egyik tagja válaszol az üzenetre.
