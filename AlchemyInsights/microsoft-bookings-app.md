# <a name="microsoft-bookings-app"></a>Microsoft Bookings alkalmazás

A Microsoft Bookings egy találkozóütemezési alkalmazás, amely segít nyomon követni a foglalásokat, a munkatársakat és az ügyfeleket. A Bookings alkalmazásban további információt vagy hibaelhárítást a [Bookings cikkeiben](https://support.office.com/article/b9c9295c-c654-4b10-b5cc-f739825fc092)talál.
