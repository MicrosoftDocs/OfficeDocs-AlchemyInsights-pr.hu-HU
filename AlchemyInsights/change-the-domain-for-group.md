# <a name="change-the-domain-for-microsoft-365-group"></a>A Microsoft 365-csoport tartományának módosítása

Ha a létrehozás során módosítani szeretné egy Microsoft 365-csoport tartományát, kövesse [a Microsoft 365-csoportok létrehozásakor használni kívánt tartomány kiválasztása](https://docs.microsoft.com/office365/admin/create-groups/choose-domain-to-create-groups)című részt.

A Microsoft 365-csoport létrehozása után jelenleg nincs mód a tartomány módosítására. Az egyetlen módja, hogy hozzon létre egy új Microsoft 365-csoport, és törölje a régit, ha szükséges (további e-mail címeket lehet hozzáadni az [Exchange Felügyeleti Központ](https://outlook.office365.com/ecp).
