# <a name="proxyaddress-incorrect"></a>A ProxyAddress helytelen

Amikor egy objektum szinkronizálva van az Azure AD-vel, az Active DirectoryproxyAddresses attribútumában megadott értékeket összehasonlítja az Azure AD-szabályokkal, majd a proxyAddresses attribútum ot feltölti az Azure AD.When an object is synchronized to Azure AD, the values that are specified in the proxyAddresses attribute in Active Directory are compared with Azure AD rules, and then the proxyAddresses attribute is populated in Azure AD. Ezért az Active Directoryban lévő objektum proxyAddresses attribútumának értékei nem egyeznek meg a proxyAddresses attribútum értékeivel az Azure AD-ben.
  
Ha többet szeretne megtudni aproxycím feltöltéséről, olvassa el [a ProxyAddress attribútum feltöltésének módja az Azure AD-ben című témakört.](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad)
