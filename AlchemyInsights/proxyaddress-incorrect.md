# <a name="proxyaddress-incorrect"></a>Helytelen ProxyAddress

Ha egy objektum szinkronizálva van az Azure AD szolgáltatással, az Active Directory proxyAddresses attribútumában megadott értékeket az Azure AD-szabályokkal hasonlították össze, majd a proxyAddresses attribútumot az Azure AD-ban tölti fel a rendszer. Lehet, hogy az Active Directoryban az objektum proxyAddresses attribútuma nem egyezik meg a proxyAddresses attribútum értékeivel az Azure AD-ban.
  
Ha többet szeretne megtudni a proxyAddress kitöltéséről, olvassa el [a proxyAddress attribútum az Azure ad-ban való feltöltéséről](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad)című témakört.
