# <a name="proxyaddress-incorrect"></a>ProxyAddress helytelen

Amikor egy objektumot szinkronizálnak az Azure AD-vel, az Active Directoryban a proxyAddresses attribútumban megadott értékeket a rendszer összehasonlítja az Azure ad szabályaival, majd a proxyAddresses attribútumot kitölti az Azure AD. Ezért előfordulhat, hogy az Active Directoryban az objektumhoz tartozó proxyAddresses attribútum értékei nem azonosak a proxyAddresses attribútum értékekkel az Azure AD szolgáltatásban.
  
Ha többet szeretne megtudni a proxycím feltöltéséről, olvassa el a [proxyAddress attribútum FELTÖLTÉSÉT az Azure](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
