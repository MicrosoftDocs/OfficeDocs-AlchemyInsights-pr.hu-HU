# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>A Microsoft Edge-házirend beállításainak megadása macOS rendszeren a JAMF használatával

Ha a Microsoft Edge (stable csatorna 81-es vagy újabb verzióját) szeretné beállítani macOS rendszeren, használja a Microsoft Edge Policy manifest-fájlt a JAMF Pro 10,19-on.

További információt [a Microsoft Edge-házirend beállításainak beállítása MacOS rendszeren a JAMF-on](https://go.microsoft.com/fwlink/?linkid=2134761)című témakörben talál.
