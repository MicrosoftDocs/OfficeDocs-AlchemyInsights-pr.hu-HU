# <a name="steps-to-create-a-sharepoint-workflow"></a>SharePoint-munkafolyamat létrehozásának lépései

- Ha nem tud SharePoint 2010-jóváhagyási munkafolyamatot hozzáadni, olvassa el A [2010-es alapértelmezett jóváhagyási munkafolyamat hozzáadása](https://docs.microsoft.com/alchemyinsights/can-t-add-default-2010-approval-workflow)című témakört.
- Munkafolyamatok létrehozása:
    - [2013-as munkafolyamat a SharePoint Designertől](https://docs.microsoft.com/sharepoint/dev/general-development/creating-a-workflow-by-using-sharepoint-designer-and-the-sharepoint-wo)
    - [2010-es munkafolyamat a SharePoint Designertől](https://support.office.com/article/introduction-to-designing-and-customizing-workflows-32c9c0bf-5e20-4f74-8b9c-d3ea79f2962b)
    - [A SharePoint-munkafolyamat bemutatása](https://support.office.com/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3)

- Munkafolyamat hibaelhárítása és hibakeresés:
    - A hibakeresési munkafolyamatok a korábbi verziókhoz hasonlítanak.  További információt a [SharePoint-munkafolyamatok hibakeresése](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows)című témakörben talál.
    - A hibaelhárításról a [SharePoint Server munkafolyamat-érvényesítési hibáinak elhárítása című témakörben talál](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio)további információt.
