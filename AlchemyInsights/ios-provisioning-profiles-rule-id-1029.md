# <a name="ios-provisioning-profiles"></a>iOS-kiépítési profilok

Az Apple iOS üzletága egy kiépítési profil koncepcióját használja, amely az alkalmazás integritásának ellenőrzésére és a profilban meghatározott szabályzatok érvényesítésére szolgál. Ezek a létesítési profilok 12 hónap után lejárnak, ezért egy új profilt kell társítani az üzembe helyezett alkalmazáshoz ahhoz, hogy az alkalmazások továbbra is fussanak.
  
A következő cikk bemutatja, hogyan hozhat létre társítást egy új létesítési profilt egy Intune-on keresztül telepített iOS LOB-alkalmazással: [Az iOS-alkalmazáskiépítési profilok használata az alkalmazások lejáratának megakadályozásához](https://docs.microsoft.com/intune/app-provisioning-profile-ios)
