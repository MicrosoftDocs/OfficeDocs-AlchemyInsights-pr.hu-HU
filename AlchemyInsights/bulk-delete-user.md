# <a name="bulk-delete-user"></a>Felhasználó tömeges törlése

Nagyszámú felhasználót távolíthat el a csoportból vesszővel tagolt értékfájl (CSV) használatával a felhasználók tömeges törléséhez. Ha tömeges felhasználókat szeretne törölni az Azure AD portál használatával, olvassa el [a Felhasználók tömeges törlése az Azure Active Directoryban](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).
