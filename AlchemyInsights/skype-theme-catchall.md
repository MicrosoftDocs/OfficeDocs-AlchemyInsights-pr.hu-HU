# <a name="skype-for-business-common-issues-and-resolutions"></a>A Skype Vállalati verzióval kapcsolatos gyakori problémák és megoldási problémák 

A Skype Vállalati verzióval kapcsolatos egyes kérdések esetén röviden fogalmazze meg a kérdését a probléma leírásához, beleértve a hibákat, a használt funkciókat stb. 

Íme néhány gyakran kért témakör:

**A Skype Vállalati verzió bejelentkezéssel kapcsolatos problémáinak elhárítása:** Először győződjön meg arról, hogy a Skype Vállalati [verzió szolgáltatás](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/servicehealth) megfelelő. Ha igen, olvassa el A [Skype Vállalati online](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/troubleshooting-sign-in-errors-for-admins#check-for-common-causes-of-skype-for-business-online-sign-in-errors) verzió bejelentkezési hibáinak elhárítása című témakört, hogy kiküszöbölje a bejelentkezési nehézségek gyakori okait.
 
**Hang- és videoproblémák elhárítása a Skype Vállalati verzióban:** Tekintse át a Skype Vállalati verzió hanggal és videóval kapcsolatos gyakori [problémáinak megoldásait.](https://support.office.com/article/Troubleshoot-audio-and-video-in-Skype-for-Business-62777bc6-c52b-47ae-84ba-a8905c3b71dc) 

**A Skype Vállalati verzió csatlakozási problémáinak elhárítása:** Tekintse át a Skype Vállalati verzió csatlakozási problémáinak [megoldásait.](https://support.office.com/article/troubleshoot-connection-issues-in-skype-for-business-ca302828-783f-425c-bbe2-356348583771)

**Kommunikáció külső Skype Vállalati verziós partnerekkel (összevonás):** A Skype Vállalati verzió beállítása a Skype Vállalati verzió külső felhasználóival [való kapcsolatfelvételhez.](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users)

**Kommunikáció a Skype-partnerekkel:** Skype Vállalati verziós felhasználók felvehetnek [Skype-partnereket.](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts)

**A Skype Vállalati verzió frissítése Microsoft Teamsre:** Tekintse [át A Microsoft Teams](https://docs.microsoft.com/microsoftteams/upgrade-start-here) frissítésének [](https://docs.microsoft.com/microsoftteams/coexistence-chat-calls-presence) első lépések és további információ a Microsoft Teams-szel való együttműködésről a Teams szervezeten belül való elfogadása során.
