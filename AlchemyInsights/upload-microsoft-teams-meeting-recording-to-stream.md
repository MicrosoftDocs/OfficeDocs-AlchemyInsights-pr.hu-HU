# <a name="upload-a-microsoft-teams-meeting-recording-to-stream-a1"></a>Microsoft Teams-értekezlet felvételének feltöltése az adatfolyamba (a1)

A Microsoft 365 legtöbb felhasználója esetén a Teams-értekezletek felvételeit a program automatikusan felveszi a Microsoft streambe az értekezlet után. Ha azonban az a1 (például a1 Plus) licenccel rendelkező felhasználók az a1  **2020-** es (például a1 Plus) licenccel rendelkező felhasználókkal kezdenek dolgozni, akkor ez a probléma megváltozik.  

További információt [a Microsoft Teams-értekezletek felvételének feltöltése az adatfolyamba](https://docs.microsoft.com/stream/portal-upload-teams-meeting-recording)című témakörben talál.
