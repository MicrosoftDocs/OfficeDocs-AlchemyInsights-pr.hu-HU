# <a name="change-your-pin"></a>A PIN-kód módosítása

Outlook.com nem használható a PIN-kód, de lehet, hogy bejelentkezik a PIN-kódot vagy a Windows Hello személyi számítógép meghajtójába. A Windows 10 PIN-kód módosítása:

1. Kattintson a Start gombra, vagy nyomja le a Windows billentyűt a billentyűzeten, majd válassza a **Beállítások** > **számlák** > **bejelentkezési beállítások**.
2. Jelölje be a **PIN-kód**alatt a **Módosítás** gombra.
3. Írja be a jelenlegi PIN-kódot, majd írja be és erősítse meg az új PIN-kód alatt.

[Tudjon meg többet a Windows Hello.](https://support.microsoft.com/help/17215/)
