# <a name="change-your-pin"></a>A PIN-kód megváltoztatása

A Outlook.com nem használ PIN-kódot, de előfordulhat, hogy bejelentkezik a SZÁMÍTÓGÉPére egy PIN-kóddal vagy a Windows Hello-nal. A PIN-kód megváltoztatása a Windows 10 rendszerben:

1. Válassza a Start gombot, vagy nyomja le a Windows billentyűt a billentyűzeten, és válassza a **Beállítások**  >  **fiók**  >  **bejelentkezési beállításai lehetőséget**.
2. Válassza a **módosítás** gombot a **PIN-kód**csoportban.
3. Adja meg a jelenlegi PIN kódját, majd írja be és erősítse meg az új PIN-kódot.

[További információ a Windows Hello szolgáltatásról.](https://support.microsoft.com/help/17215/)
