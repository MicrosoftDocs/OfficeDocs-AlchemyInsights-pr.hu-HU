# <a name="service-principal-key"></a>Egyszerű szolgáltatáskulcs

Az egyszerű szolgáltatásnév fő hitelesítő adatainak kezeléséhez az alábbi cikkekben talál útmutatást:

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) – Egy egyszerű szolgáltatásnév kulcs hitelesítő adatainak lekérte a PowerShell használatával.

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) – Új kulcs hitelesítő adatok létrehozása egyszerű szolgáltatásnévhez a PowerShell használatával.

- [Remove-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) – Kulcs hitelesítő adatok eltávolítása egyszerű szolgáltatásnévből a PowerShell használatával.
