# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Feltételek vagy speciális képletek használata a forgalomban

A Microsoft flow-ban testre szabhatja a folyamatokat úgy, hogy csak a feltétel teljesülése esetén végezze el a feladatokat. További információt a [feltétel hozzáadása egy adatfolyamhoz](https://go.microsoft.com/fwlink/?linkid=872112)című témakörben talál.
  
Ha speciális képletet szeretne létrehozni, olvassa el [a munkafolyamat-definíciós nyelvi séma az Azure Logic apps rendszerhez](https://aka.ms/logicexpressions)című témakört.
