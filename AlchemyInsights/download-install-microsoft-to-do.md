# <a name="how-to-download-and-install-microsoft-to-do"></a>A Microsoft to do letöltése és telepítése

A Microsoft to do app a Win32, az iOS, az Android és a Mac App alkalmazásban letölthető a megfelelő áruházból.

Ha korlátozott hozzáférése van a bérlői webhelyéhez, olvassa el [https://docs.microsoft.com/microsoft-store/distribute-offline-apps](https://docs.microsoft.com/en-us/microsoft-store/distribute-offline-apps) az offline telepítő használata című témakört.

Ha a **munkahelyi** fiók használatával szeretne letölteni egy alkalmazást az áruházból, az alkalmazást közzé kell tennie a cég **privát áruházában.** További információért olvassa el a [https://docs.microsoft.com/en-us/microsoft-store/app-inventory-management-microsoft-store-for-business#private-store-availability](https://docs.microsoft.com/microsoft-store/app-inventory-management-microsoft-store-for-business#private-store-availability)
