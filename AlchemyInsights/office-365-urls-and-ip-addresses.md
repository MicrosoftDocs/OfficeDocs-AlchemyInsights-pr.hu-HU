# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Microsoft 365 URL-címek és IP-címtartományok

A Microsoft 365-nek végpontnak *nevezett*IP-címeken keresztül internetkapcsolatra van szüksége.
Az elérendő végpontok listáját a Microsoft [365 URL-címei és IP-címtartományai című témakörben található.](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

A Microsoft 365 IP-cím és URL-cím webszolgáltatás segítségével jobban azonosíthatja és megkülönböztetheti a Microsoft 365 hálózati forgalmát, így könnyebben értékelheti, konfigurálhatja és naprakészen tarthatja a módosításokat. Lásd: [Office 365 IP-cím és URL-webszolgáltatás.](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service)
