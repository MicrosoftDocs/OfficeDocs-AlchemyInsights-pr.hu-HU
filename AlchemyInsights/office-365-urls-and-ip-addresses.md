# <a name="microsoft-365-urls-and-ip-address-ranges"></a>A Microsoft 365 URL-címei és IP-címtartományok

A Microsoft 365 a *végpontok*nevű meghatározott IP-címeken keresztül kapcsolódik az internethez.
A megtekinteni kívánt végpontok listáját a [Microsoft 365 URL-címei és IP-címtartományok](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) című témakörben találhatja meg. 

A Microsoft 365 IP-címe és URL-címe webszolgáltatással jobban azonosíthat és megkülönböztetheti a Microsoft 365 hálózati forgalmát, így megkönnyítheti a módosítások kiértékelését, konfigurálását és naprakész állapotba helyezését. Lásd: [Microsoft 365 IP-cím és URL-webszolgáltatás](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
