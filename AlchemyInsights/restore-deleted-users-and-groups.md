# <a name="restore-deleted-users-and-groups"></a>Törölt felhasználók és csoportok visszaállítása

Véletlenül törölt felhasználó visszaállításáról a [nemrég törölt felhasználó visszaállítása az Azure Active Directoryhoz](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-users-restore?context=azure/active-directory/users-groups-roles/context/ugr-context)című témakörben olvashat.

A törölt csoportok visszaállításáról a [törölt Microsoft 365-csoport visszaállítása az Azure Active Directoryban](https://docs.microsoft.com/azure/active-directory/users-groups-roles/groups-restore-deleted)című témakörben olvashat.
