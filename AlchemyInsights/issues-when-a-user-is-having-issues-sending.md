# <a name="a-user-is-having-issues-sending-email-messages"></a>A felhasználónak problémái vannak az e-mail üzenetek küldésével

Ez úgy néz ki mint önnek van egy felad folyik kérdés. Van diagnosztikája, amely segít azonosítani sok közös levéláramlási problémát. Kattintson az alábbi, **diagnosztika futtatása** elemre.
