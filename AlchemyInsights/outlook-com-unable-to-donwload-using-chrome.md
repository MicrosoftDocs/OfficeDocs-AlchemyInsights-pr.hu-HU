# <a name="cant-download-files-in-chrome"></a>Nem lehet letölteni a fájlokat a Chrome-ban

**Kiadás** Tudomásunkra került egy olyan probléma, amely megakadályozza, hogy egyes ügyfelek fájlokat töltsenek le a Google Chrome böngésző használata során. 

**Kerülő megoldás** Használja a [Microsoft Edge-et](https://www.microsoft.com/windows/microsoft-edge) vagy egy másik webböngészőt.
További információ a [fájlok és Outlook.com mellékletek letöltéséről.](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy)
